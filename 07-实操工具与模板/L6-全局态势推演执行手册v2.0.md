# L6 全局态势推演 × 论持久战式沙盘 — 执行手册 v2.0

> **L5 回答"该不该做"，L6 回答"所有相关方将如何行动，战局如何演化，标的在真实未来场景中是什么状态"。**
> **L6 的思维范式源自毛泽东《论持久战》：全面分析交战各方的全部要素，从力量对比的动态演化中预见未来。**

---

## 思想根基：为什么用《论持久战》的方法做投资推演？

> *"料敌如神"的背后，其实就是高层次思维带来的绝对优势。*
> *——对《论持久战》的核心评价*

《论持久战》之所以能"穿越时空"，不在于它预测了具体的战役胜负，而在于它提供了一套**全面分析对立双方全部要素、从力量消长的动态演化中预见最终结局**的思维方法。

这套方法的核心支柱，恰恰是投资推演最需要的能力：

| 《论持久战》方法论 | 投资推演的对应 |
|---|---|
| **全面的力量对比模型**（四维：现实战力/发展趋势/战略纵深/外部支援） | 不只看标的公司本身，而是扫描所有相关方的全部要素进行对比 |
| **现实维度 vs 趋势维度的区分** | 区分"决定当前股价的因素"与"决定最终胜负的因素"，两者的差距就是投资机会 |
| **阶段性推演**（防御→相持→反攻） | 给标的的未来演化"描画一个轮廓供指导，事实再校正之" |
| **主要矛盾识别** | 在众多分析维度中，找出那个"一切问题迎刃而解"的关键变量 |
| **群众路线 / 兵民是胜利之本** | 分析企业的用户/员工/生态伙伴——这些"不在资产负债表上"的力量 |
| **驳亡国论与速胜论** | 拒绝"必涨"和"必跌"的极端预测，基于全面分析给出概率化判断 |

**v2.0 的核心升级**：从 v1.0 的"红蓝军辩论"升格为**"全相关方力量对比 → 行为驱动因素拆解 → 动态推演 → 真实未来场景"**的完整投资推演体系。

## 定位：L5 vs L6 v2.0

| 维度 | L5 买方决策 | L6 v2.0 全局态势推演 |
|------|-----------|---------------------|
| **核心问题** | 该不该做？值不值得？ | 所有相关方将如何行动？战局如何演化？标的在真实未来是什么状态？ |
| **思维模式** | 判断者 (Decision Maker) | 指挥官 + 预言家 (War Commander + Prophet) |
| **分析范围** | 以标的公司为中心 | 以**全部相关方的力量对比**为中心 |
| **时间维度** | 快照式（基于当前信息的最佳判断） | 动态演化式（从T+0推演到T+5的阶段性变化） |
| **方法论根基** | 多框架交叉验证 | **《论持久战》四维力量对比 + 阶段推演 + 主要矛盾分析** |
| **类比** | CIO 向合伙人汇报 | 参谋长向统帅做战场态势推演 |

**L5 的盲区，正是 L6 v2.0 要填的**：
1. L5 分析标的公司本身，但没有**穷尽性地梳理所有相关方及其行为驱动因素**
2. L5 做了情景分析，但没有从**各方力量消长的动态**中推导出情景的概率
3. L5 给出了"买入"建议，但没有回答"买入之后，**各方会如何反应**，战局会如何演化"
4. L5 对竞争格局做了分析，但没有追问每个竞争对手的**内部主要矛盾**和**行为约束**
5. L5 是分析师视角，L6 v2.0 要升维到**战略家视角——不仅看这一场仗，而是看整个战争**

### 模块零：问题的提起 (What Are We Answering?)

> **《论持久战》开篇即问**："战争的过程究竟会要怎么样？能胜利还是不能胜利？能速胜还是不能速胜？"
> **投资推演同样必须先明确**：围绕这个标的，市场上有哪些观点？我们要回答什么？

#### 0.1 识别市场上的三类观点

对每一个投资标的，市场上通常存在三类典型观点，对应《论持久战》中驳斥的两种极端和自身的结论：

| 类型 | 《论持久战》对应 | 投资推演对应 | 典型表述 |
|------|---------------|------------|---------|
| **速胜论** | 认为中国可速胜日本 | 极度乐观派 | "这个公司将快速增长，明年翻倍" |
| **亡国论** | 认为中国必亡 | 极度悲观派 | "这个公司要完了，逻辑崩塌" |
| **持久论** | 持久战，最终胜利 | 理性分析派 | "竞争将长期化，需阶段性评估" |

**执行动作**：
1. 收集并整理市场上最有代表性的"速胜论"观点（最看多的卖方/KOL说了什么？逻辑是什么？）
2. 收集并整理最有代表性的"亡国论"观点（最看空的声音说了什么？做空报告的核心逻辑？）
3. 明确我们要回答的核心问题：**基于全部相关方的力量对比和动态推演，这个标的的未来轮廓是什么？**

### 模块一：全局相关方扫描 (Comprehensive Stakeholder Mapping)

> **《论持久战》不只分析中日两国，而是扫描"敌国方面""国际方面""国内方面"的全部力量。**
> **"全民族的力量团结起来""全世界人民都关心这个战争"——穷举所有影响战局的力量。**

#### 1.1 六层相关方扫描模型

对标的公司所处的"战场"进行穷举性扫描：

```
第一层：内部力量 (Internal Forces)
  ├─ 管理层/创始人（指挥官）
  ├─ 核心员工/研发团队（军队主力）
  ├─ 董事会/大股东（后方统帅部）
  └─ 中层管理者（基层军官）

第二层：竞争力量 (Competitive Forces)
  ├─ 直接竞争对手（1-3家核心"敌军"）
  ├─ 潜在进入者（可能入局的新力量）
  ├─ 替代品/颠覆者（不同维度的威胁）
  └─ 同业跟随者（中小竞争者群体）

第三层：产业链力量 (Value Chain Forces)
  ├─ 上游供应商（原材料/技术/组件）
  ├─ 下游客户/用户（付费的"群众"）
  ├─ 渠道伙伴/分销商（连接供需的"邮路"）
  └─ 互补品提供者（生态内的协作方）

第四层：资本市场力量 (Capital Market Forces)
  ├─ 机构投资者/大基金（"盟军"或"敌军"）
  ├─ 卖方分析师/评级机构（"舆论阵地"）
  ├─ 做空机构（专业"红军"）
  └─ 散户投资者群体（"市场群众"）

第五层：监管与政策力量 (Regulatory Forces)
  ├─ 行业监管机构（直接管辖方）
  ├─ 宏观经济政策制定者（利率/财政/产业政策）
  ├─ 国际贸易/地缘政治方（贸易壁垒/制裁）
  └─ 地方政府/产业园区（具体落地环境）

第六层：社会与环境力量 (Societal Forces)
  ├─ 媒体/舆论/KOL（"宣传阵地"）
  ├─ 行业协会/标准制定机构
  ├─ 学术/科研机构（技术突破的源头）
  └─ 社会趋势/消费文化/人口结构变化
```

对每一个被识别的关键相关方，填写以下登记信息：

| # | 层级 | 相关方名称 | 角色定位 | 对标的的影响方向 | 影响力等级 | 当前立场 |
|:---:|------|---------|---------|--------------|---------|---------|
| 1 | 第X层 | 具体名称 | 敌方/友方/中立/不确定 | 利好/利空/双向 | 高/中/低 | 支持/对抗/观望 |
| 2 | | | | | | |
| ... | | | | | | |

**重要原则**：
- **穷举优于精选**：宁可多列不可遗漏，之后再做重要性排序
- **特别关注"隐性相关方"**：如行业标准制定者、关键专利持有者、前员工群体等
- **识别"变节可能"**：哪些当前的友方可能转为对手？哪些对手可能成为盟友？

### 模块二：四维力量对比 (Four-Dimensional Force Comparison)

> **这是整个 L6 v2.0 的分析核心，直接来源于《论持久战》的分析方法。**
>
> 毛泽东分析中日两国，用了四个维度：**军力经济力政治组织力**（现实战力）、**退步/进步**（发展趋势）、**国土人力物力**（战略纵深）、**国际支援**（外部力量）。
>
> 关键洞见：**第一维度决定当前谁占上风，后三个维度决定最终谁赢。两者的差距，就是"持久战"存在的原因，也是投资机会存在的原因。**

#### 2.1 四维分析框架

**对标的公司与其核心竞争对手（1-3家），逐一进行四维对比：**

##### 维度一：现实战力（决定"当前谁赢"）

> *"日本是一个强的帝国主义国家，它的军力、经济力和政治组织力在东方是一等的。"*
> *——这是对现实的承认，不粉饰。*

| **综合评价** | 强/中/弱 | 强/中/弱 | 强/中/弱 | 强/中/弱 |

> *"日本的战争是退步的和野蛮的……中国的战争是进步的和正义的。"*
> *——"退步"不只是道德判断，而是指其模式与时代趋势背离。"进步"意味着其模式与趋势一致。*

### 模块三：各方行为驱动因素深度拆解 (Behavioral Drivers Analysis)

> **《论持久战》不只看"敌人有多强"，更分析敌人为什么这样做、受什么约束、有什么内部矛盾。**
> *"日本社会经济的帝国主义性，就产生了日本战争的帝国主义性，它的战争是退步的和野蛮的。"*
> *——行为是由内在性质决定的。理解了性质，就能预测行为。*

### 模块四：主要矛盾识别与矛盾转化预判 (Principal Contradiction)

> **"在复杂的事物的发展过程中，有许多的矛盾存在，其中必有一种是主要的矛盾，由于它的存在和发展，规定或影响着其他矛盾的存在和发展。"**
> **"捉住了这个主要矛盾，一切问题就迎刃而解了。"**

#### 4.1 矛盾清单梳理

从前三个模块的分析中，提炼出标的公司面临的所有核心矛盾：

|:---:|---------|---------|------------|---------|
| 1 | 增长 vs 盈利 | 市场份额扩张 vs 利润率维持 | [哪面占支配] | 高/中/低 |
| 2 | 创新 vs 守成 | 新业务投入 vs 现金牛维护 | | |
| 3 | 集权 vs 分权 | 总部控制 vs 地方自主 | | |
| 4 | 高端 vs 下沉 | 高利润客群 vs 广阔大众市场 | | |
| 5 | 自研 vs 外购 | 技术自主 vs 效率优先 | | |
| ... | | | | |

#### 4.2 主要矛盾判定

从上述清单中，选出**一个**最核心的主要矛盾（判定标准：它的解决/激化将连带改变其他所有矛盾的走向）。

```markdown
## 主要矛盾判定

### 当前主要矛盾
[X] vs [Y]

### 判定理由
1. 它为什么是"主要"的？（规定或影响着哪些次要矛盾？）
2. 管理层是否识别了这个主要矛盾？资源配置是否聚焦于此？
3. 如果这个矛盾得到解决，哪些次要矛盾将自动化解？

### 矛盾主要方面的当前状态
当前，[X/Y] 占支配地位，表现为 [具体现象]。

### 矛盾转化预判（最关键的预测）
> "事物的性质，主要地是由取得支配地位的矛盾的主要方面所规定的。"
> "这种情形不是固定的，矛盾的主要和非主要的方面互相转化着。"

**转化的触发条件**：什么因素会导致矛盾主要方面发生翻转？
**转化的时间窗口**：大约在什么时候？
**转化后的新性质**：一旦翻转，公司/行业将呈现什么新特征？

### 投资含义
- 如果矛盾**按当前方向**继续发展 → [对投资意味着什么]
- 如果矛盾主要方面**发生翻转** → [对投资意味着什么]
- 翻转的概率估计 → [X%]
```

### 模块九：执行兵法 (Execution Playbook)

> **《论持久战》不只预判战争走向，更给出了具体的作战方针。**
> **"歼灭战/消耗战""运动战/游击战/阵地战"——不同阶段用不同的打法。**

#### 9.1 建仓战术

| 阶段 | 行动 | 仓位 | 触发条件 | 《论持久战》对应 |
|------|------|------|---------|--------------|
| **侦察仓** | 小额试探 | 组合的 1-2% | 完成 L5 分析，逻辑初步验证 | "游击战"——试探敌情 |
| **底仓建立** | 核心建仓 | 目标仓位的 50% | L6 净置信度 > 65% | "运动战"——集中力量打歼灭战 |
| **加仓信号** | 逢低/催化剂加仓 | 至目标仓位 100% | 关键催化剂兑现 / 无理由下跌20%+ | "乘敌之隙" |
| **超配信号** | 突破目标仓位 | 最高 150% | 净置信度 > 80% + 催化剂密集兑现 | "战略反攻"——全面投入 |

#### 9.2 作战方式选择（v2.0新增）

> **"运动战，游击战，阵地战是持久战的三种战争形式。"**
> **不同阶段，作战方式的主次配合不同。**

| 标的所处阶段 | 主要作战方式 | 辅助方式 | 投资对应 |
|------------|-----------|---------|---------|
| **战略防御期** | 游击战（小仓位灵活进出） | 运动战（把握局部确定机会） | 以侦察仓为主，逢大跌试探加仓，保持灵活 |
| **战略相持期** | 运动战（把握确定性机会加仓） | 阵地战（核心仓位不动摇） | 底仓+运动战式加减仓，利用波动积累 |
| **战略反攻期** | 阵地战（坚定持有核心仓位） | 运动战（利用回调加仓） | 满仓持有，让利润奔跑 |

**关键原则**：
- **"有把握就坚决打，无把握就坚决不打"**——把握不大的机会果断回避
- **"避免赌博式战略决战"**——不在极端不确定时重仓All-in
- **"以时间换空间"**——如果趋势维度占优但现实维度暂时落后，用时间等待力量转化

#### 9.3 持仓铁律（不可违背）

| 铁律 | 内容 | 理由 |
|------|------|------|
| **单一标的上限** | 不超过组合的 X%（用户设定） | 防止单点失败毁灭组合 |
| **时间框架锁定** | 考核周期 ≥ 3年 | 持久战不能用短期胜负衡量 |
| **逻辑未变不减仓** | 股价波动非减仓理由 | 除非红军论点被证实 |
| **逻辑崩塌则无条件清仓** | Kill-Switch 触发 = 24h 内退出 | 不恋战不幻想 |
| **趋势维度变化要重评** | D2/D3/D4任一发生重大变化 → 重新做L6 | "进步/退步"是最根本的判断 |

#### 9.4 Tripwire 系统（预设绊线）

| 绊线类型 | 具体触发条件 | 到达后的行动 | 检查频率 |
|---------|------------|------------|---------|
| **加仓绊线** | 股价跌破 $X（逻辑未变前提） | 自动加仓 10% | 每日 |
| **减仓绊线** | 估值达到目标价的 80%+ | 减仓至标准仓位 | 每周 |
| **退出绊线** | 红军任一致命论点被证实 | 24h 内清仓 | 每季报/随时 |
| **重评绊线** | 主要矛盾发生意外转化 / 关键相关方行为异常 | 重新执行 L6 推演 | 半年度 |
| **止盈绊线** | 仓位盈利达 100%+ | 减仓 50% 锁利，余额让利润奔跑 | 每月 |
| **阶段转换绊线**（v2.0新增） | 标的从"防御期"进入"相持期"或"反攻期" | 调整作战方式的主次配合 | 每季度 |

#### 9.5 战后复盘协议 (After-Action Review)

**预设复盘时间点**（在建仓时就锁定）：

| 复盘时间 | 审查内容 | 行动标准 |
|---------|---------|---------|
| **T+3个月** | 初始假设是否仍然成立？ | 红军论据有无新证据？ |
| **T+6个月** | 催化剂是否按时间表兑现？ | 未兑现的催化剂 → 下调置信度 |
| **T+12个月** | **四维力量对比是否发生变化？** | 任一维度翻转 → 重做推演 |
| **每次季报后** | 财务数据是否验证多头逻辑？ | KPI 达标率 < 70% → 减仓 |
| **异常波动后** | 波动原因是噪音还是信号？ | 信号 → 重新推演；噪音 → 按兵不动 |
| **每半年** | **各相关方行为是否偏离预判？** | 重要偏离 → 更新行为分析 |
