# Agent OS — 基金经理认知守护系统

> **你的存在不是为了替交易者赚钱，而是为了防止他亏掉本不该亏的钱。**

## 这是什么

Agent OS 是一个**认知守护框架**——不是一个需要运行的程序，而是一套让 AI 按照交易纪律来思考的规则系统。

打开这个文件夹，VS Code 里的 AI 就会自动读取这些规则，变成你的：
- 📋 盘前分析师
- 🔍 盘中守护者
- 🛡 风控审查员
- 📖 盘后复盘导师

## 如何使用

### 第一步：打开文件夹
在 VS Code 中打开 `agent_os` 文件夹。

### 第二步：开始对话
在 VS Code 的 AI 对话框（Copilot Chat / Claude）中直接对话。

### 第三步：用自然语言操作

| 你说 | 系统做 |
|------|--------|
| **"开盘"** | 执行盘前研判，生成晨间简报，设定边界条件 |
| **"看看贵州茅台"** | 帮你建立判断，要求你写证伪条件 |
| **"想买600519"** | 生成 Decision Brief，执行风控审查 |
| **"确认 d001"** | 批准并记录交易 |
| **"更新"** | 盘中监控，检查判断是否被市场破坏 |
| **"状态"** | 输出当前系统状态总览 |
| **"风控"** | 输出风控仪表板 |
| **"收盘"** | 执行盘后复盘，四维评分，归档 |
| **"平仓 t001"** | 记录平仓，计算盈亏 |
| **"回顾2月7日"** | 调取历史记录 |

### 第四步：保持纪律
AI 会在你想冲动的时候逼你停下来思考。**不要跳过它的提问。**

## 架构设计

```
传统方案                          Agent OS
─────────                        ─────────
写爬虫抓新闻     ────→    AI 模型自身就有市场知识
调 API 拿数据    ────→    用户在对话中告知行情
搭后端服务器     ────→    打开 VS Code 即可
需要 API Key     ────→    只需 VS Code 内置 AI
```

**核心思路：让 Claude / GPT 这样的强模型充当"智能数据层"。**
它们的训练知识涵盖全球金融市场信息、宏观经济、行业逻辑、交易心理学，
本身就是最好的分析引擎。不需要再费力写 Python 爬虫。

## 文件结构

```
agent_os/
├── .github/
│   └── copilot-instructions.md  ← AI 自动读取的系统指令
├── AGENT_OS.md                   ← 完整系统设计文档
├── README.md                     ← 你正在看的使用指南
├── config.json                   ← 个人配置（名字、初始资金等）
│
├── workflow/                     ← 工作流定义（AI 按此执行）
│   ├── 01_盘前研判.md
│   ├── 02_判断管理.md
│   ├── 03_盘中监控.md
│   ├── 04_决策执行.md
│   ├── 05_风控审查.md
│   └── 06_盘后复盘.md
│
├── state/                        ← 运行状态（AI 在对话中读写）
│   ├── judgments.json             判断列表
│   ├── decisions.json            决策记录
│   ├── trades.json               交易记录
│   ├── portfolio.json            持仓状态
│   └── daily_boundary.json       边界条件
│
├── rules/
│   └── risk_config.json          ← 风控硬规则（不可绕过）
│
└── vault/                        ← 历史归档（按日期保存）
```

## 配置个性化

编辑 `config.json`：
- 填入你的名字
- 设置初始资金
- 设置交易风格、关注板块

编辑 `rules/risk_config.json`：
- 调整仓位限制
- 调整止损范围
- 调整日损限额

## 设计哲学

1. **模型即数据源** — AI 自身的训练知识就是最好的市场信息源
2. **对话即操作界面** — 不需要写代码，自然语言就是最好的接口
3. **文件夹即系统** — 无需部署、无需配置环境、无需 API Key
4. **纪律即策略** — 真正的 alpha 不在信号里，在于执行纪律
