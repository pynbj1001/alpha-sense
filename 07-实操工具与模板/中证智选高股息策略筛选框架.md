> 基于《中证智选高股息策略指数编制方案》(932305) 提炼，用于高股息个股筛选
> 原始文件：`01-价值投资体系/中证智选高股息策略指数编制方案.pdf`

---

## 一、策略核心逻辑

**一句话定义**：选取连续分红、现金分红预案股息率高、波动率低的股票，构建高股息+低波动的防御型组合。

**核心公式**：

$$\text{加权因子} = \frac{\text{现金分红预案股息率}}{\text{过去一年波动率}}$$

> 这意味着：**同等股息率下偏好波动率更低的标的**，本质是对"单位风险下的股息收益"进行排序。

---

## 二、筛选条件清单（逐层过滤）

### 第 1 层：样本空间
- 属于**中证全指指数**的样本空间（沪深两市，剔除 ST、*ST、暂停上市等）

### 第 2 层：流动性筛选
- [x] 过去一年**日均成交金额**排名位于样本空间**前 90%**
- 目的：剔除流动性极差的微小盘股

### 第 3 层：分红连续性筛选
- [x] **过去三年连续现金分红**
- [x] **已披露现金分红预案**，且预案中合计每股税前现金股利 > 0
- 目的：筛选出有稳定分红意愿和能力的公司

### 第 4 层：股息率排序选股
- 根据上市公司**已披露的现金分红预案**计算股息率
- 按**预案股息率降序排列**
- 选取排名前 **50 只**证券

### 第 5 层：加权方式（构建组合时）
- 采用 **股息率/波动率** 加权
- 单个样本权重**上限 10%**
- 等价于：偏好"高股息+低波动"双因子的股票

---

## 三、个股筛选实操步骤

以下为将指数编制方案转化为个人选股流程的操作指引：

### Step 1：拉取分红数据
```python
import akshare as ak

# 获取A股分红数据
# 检查过去三年是否连续现金分红
# 获取最新已披露的分红预案
df_dividend = ak.stock_history_dividend()
```

### Step 2：计算预案股息率
$$\text{预案股息率} = \frac{\text{每股税前现金股利（预案）}}{\text{当前股价}} \times 100\%$$

### Step 3：计算过去一年波动率
$$\sigma = \text{std}(R_{\text{daily}}) \times \sqrt{252}$$

### Step 4：计算综合得分并排序
$$\text{Score} = \frac{\text{预案股息率}}{\sigma_{\text{年化}}}$$

按 Score 降序排列，选取 Top 50。

### Step 5：权重约束
- 初始权重 ∝ Score
- 归一化后，将超过 10% 的个股截断至 10%，多余部分按比例分配给其余个股

## 四、关键设计要点解读

| 设计要素 | 具体规则 | 投资含义 |
|---|---|---|
| **基于预案而非实际分红** | 使用已披露分红预案计算股息率 | 前瞻性更强，反映管理层最新分红意愿 |
| **连续三年分红** | 过去三年均有现金分红 | 过滤"一次性分红"陷阱，确保分红可持续性 |
| **股息率/波动率加权** | 非等权、非市值加权 | 双因子策略：高股息 × 低波动 |
| **单只上限 10%** | 权重不超过 10% | 防止集中度风险 |
| **年度调整** | 每年 5 月第六个交易日调整 | 对应年报+分红预案披露节奏 |
| **50 只成分股** | 固定数量 | 保证分散度，同时聚焦高股息标的 |

## 五、筛选时需注意的陷阱

### ⚠️ 高股息陷阱识别
1. **周期性利润导致的虚高股息率**：大宗商品/周期股在利润高峰期股息率极高，但不可持续
2. **一次性特殊分红**：本框架要求"连续三年分红"已做部分过滤，但仍需检查分红的趋势
3. **分红预案≠实际分红**：预案可能被调整或取消（概率低但存在）
4. **股价暴跌导致的被动高股息率**：公司基本面恶化时股价下跌推高股息率

### ✅ 配合使用的补充检查
- **盈利稳定性**：ROE 是否稳定在合理区间？（配合市赚率框架）
- **自由现金流覆盖率**：FCF ÷ 分红总额 > 1.5x 为佳
- **负债率**：分红是否建立在高杠杆基础上？
- **行业分散度**：避免全部集中在银行/煤炭等单一行业

## 六、与知识库其他框架的协同

| 配合框架 | 协同方式 |
|---|---|
| **市赚率框架** | 用 PR = PE/ROE 进一步筛选"高质量+高股息"标的 |
| **巴菲特护城河** | 对 Top 50 中的标的做护城河评估，优选有竞争壁垒的高股息股 |
| **卡拉曼安全边际** | 以股息率隐含的估值作为安全边际的一个维度 |
| **中证红利质量指数** | 交叉对比两个指数的成分股重合度，重合标的质量更高 |
| **龙头加减仓体系** | 筛选出标的后，用龙头体系 v3.1 做仓位管理和买卖点择时 |
| **钟摆指数** | 在市场情绪低迷（钟摆偏恐惧）时加大高股息配置比例 |

### 三、Kill-Switch 红线（本框架专属）

| 触发条件 | 严重程度 | 行动 |
|---------|---------|------|
| 分红不可持续（FCF<分红）| 致命 | 排除 |
| 周期性利润导致虚高股息 | 致命 | 排除 |
| 负债率过高（>70%）| 严重 | 排除 |
| 行业集中度>50% | 严重 | 分散化 |

### 四、领先指标监控

| 指标 | 监控频率 | 预警阈值 | 含义 |
|------|---------|---------|------|
| 股息率变化 | 季度 | 下降超过30% | 分红可能减少 |
| 波动率变化 | 月度 | 显著上升 | 风险加大 |
| FCF/分红 | 季度 | 低于1.0 | 分红不可持续 |
| 盈利稳定性 | 年度 | ROE波动>30% | 周期性风险 |

### 五、适用范式标签
- **正范式适配**：现金回报防御、低估催化
- **负范式检测**：深度价值陷阱、周期陷阱、杠杆驱动
- **最佳使用场景**：低利率环境、防御型配置、高净值投资者
- **不适用场景**：高通胀环境、货币紧缩周期

### 六、可证伪条件模板
1. 如果 **FCF覆盖率降至1.0以下**，说明 **分红不可持续**，需 **卖出**
2. 如果 **行业集中度超过50%**，说明 **风险过度集中**，需 **分散化**
3. 如果 **ROE波动超过30%**，说明 **周期性风险高**，需 **降低仓位**
