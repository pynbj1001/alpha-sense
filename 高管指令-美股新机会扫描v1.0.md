# 高管指令｜美股新机会扫描（v1.0）

> 生成日期：2026-02-20  
> 触发背景：用户当前持仓/关注美股龙头逆向机会+特斯拉，希望系统化扩展覆盖范围，寻找更多符合个人风格的高概率机会。

---

## 一、决策目标

在已覆盖标的（MSFT、TSLA、MSTR、GOOGL、MU、TSM、NET）基础上，向外系统扩展，  
**找出 5～8 只** 当前最值得跟踪的美股机会，要求：
- 符合用户"逆向龙头"或"成长拐点"偏好
- 当前估值具备安全边际或隐含显著超额回报（TSR decomp）
- 商业模式清晰，可在 30 秒内解释清楚

---

## 二、用户风格约束（硬约束）

| 约束维度 | 具体要求 |
|---------|---------|
| 风格 | 逆向优先——偏好已出现较大跌幅、市场情绪悲观的龙头 |
| 能力圈 | 科技/AI基础设施/平台/消费科技，不涉及医药/矿产/小盘/黑箱 |
| 护城河门槛 | 至少拥有网络效应/成本优势/转换成本之一 |
| 安全边际 | 卡拉曼标准：当前 P/EPV < 1.5x，或安全边际 ≥ 20% |
| 时间框架 | 中长期（1～3年），不以短期交易为主 |
| 排除标的 | 已在自选列表中的：MSFT、GOOGL、MU、TSM、POP MART、NET、CNOOC |

---

## 三、最高原则（按优先级）

1. **数据先行** > 叙事先行：所有结论必须基于实时获取的财务/价格数据
2. **逆向质量** > 便宜便宜再便宜：要求质量 + 低估，而非纯低估
3. **可证伪** > 好听：每个机会必须有明确的失效条件
4. **集中** > 分散：宁可 5 只高置信度，不要 20 只低密度机会

---

## 四、候选池生成逻辑

### 4.1 初筛宇宙（30支 → 压缩至 5～8 支）

分三类入池：

**A. 逆向龙头类**（从历史高点跌幅 >20% 的科技/平台龙头中挑）
- NVDA（DeepSeek冲击后回调）
- META（AI资本开支担忧）
- AMZN（AWS定价+关税担忧）
- AAPL（iPhone增长放缓）
- UBER（自动驾驶颠覆担忧）
- SNAP / PINS（广告小盘，排除）

**B. 成长拐点类**（估值已修正、基本面正在改善的成长股）
- TSLA（FSD拐点+能源业务被低估）
- NOW（ServiceNow，企业AI落地最快平台之一）
- SNOW（Snowflake，AI数据平台估值大幅修正）
- PLTR（Palantir，AI+国防，争议大）
- ASML（芯片设备垄断，周期底部附近）

**C. 平台生态类**（具备网络效应飞轮，当前增长被错误定价）
- SHOP（Shopify，电商基础设施）
- SPOT（Spotify，音频平台，盈利拐点）
- DUOL（Duolingo，AI学习平台）

### 4.2 第一轮量化筛选（Python执行）

拉取以下指标，构建机会矩阵：

| 指标 | 筛选目的 |
|-----|---------|
| 股价距52周高点跌幅 (%) | 逆向程度衡量 |
| 市盈率 (trailing PE) | 估值横向比较 |
| 市赚率 PR = PE / ROE | 综合质量估值比 |
| 市净率 PB | 资产估值锚 |
| ROE (TTM) | 资本质量 |
| 毛利率 | 商业模式定价权 |
| 营收增速 YoY | 成长动能 |
| 自由现金流收益率 | 安全边际（现金流维度） |
| 市值 | 流动性确认 |

### 4.3 第二轮框架打分（贝叶斯+市赚率+护城河）

对第一轮留下的候选标的，执行：

- **市赚率 PR 评分**：PR < 1.5 = 优秀，1.5~3.0 = 合理，>3.0 = 昂贵
- **护城河快速打分**：网络效应/成本优势/转换成本/无形资产/规模效应 各1分，满分5分
- **贝叶斯拐点概率**：当前市场悲观假设是否已过度定价？（主观打分 0~100%）
- **Kill-Switch 检查**：逐一排查7项红线

---

## 五、执行步骤

### Step 1：数据拉取（Python - yfinance）
拉取候选池全部标的的实时财务数据，构建上述指标矩阵。

### Step 2：逆向程度排名
按"距52周高点跌幅"排名，识别当前市场最悲观的标的。

### Step 3：质量过滤
排除 ROE < 10%、毛利率 < 35%、营收增速 < 8% 的标的（除非有明确周期底部理由）。

### Step 4：PR矩阵排名
综合市赚率+自由现金流收益率，产出"質優 × 位低"象限图。

### Step 5：最终机会榜（前5～8）
输出每只标的的：
- 一句话投资论点
- 当前估值快照（PE/PB/PR/FCF yield）
- 市场悲观假设是什么（可证伪条件）
- 贝叶斯拐点概率（主观）
- 建议优先级（高/中/低）

- 每只标的均有明确的可证伪失效条件
- 报告生成后追加到 `stock_tracker.py` 自选列表（征得用户确认）
