
一、原始市赚率：丁宁这玩意到底是啥？
先把记号说清楚（我统一用“百分数口径”的 ROE，比如 20% 就写成 20）：
股价：P
每股收益（TTM）：EPS
每股净资产：BPS
市盈率：PE = P / EPS
市净率：PB = P / BPS
ROE%：比如 20% 就记作 20
1. 核心定义公式
丁宁的市赚率（PR）核心定义是：
市赚率 PR = 市盈率 PE ÷ ROE（%）
也就是：
PR = PE / ROE%
如果一家公司 PE=15，ROE=15%，那 PR=15/15=1
直觉理解：
PE 告诉你“为了 1 块钱利润，你愿意出多少倍价格”；
ROE 告诉你“每 1 块净资产，一年能赚几个百分点”；
PE / ROE% 就是在问：
“为了每 1 个百分点的 ROE，市场愿意给出多少倍市盈率？”
所以丁宁的基准是：
PR ≈ 1：大致合理
PR < 1：便宜，越低越有折扣
PR > 1：偏贵，越高溢价越大

2. 等价“第二公式”：用 PB 和 ROE 算 PR
从会计恒等式出发（注意 ROE 用小数）：
ROE = EPS / BPS
PB = P / BPS = (P / EPS) × (EPS / BPS) = PE × ROE(小数)
把 ROE(小数) = ROE% / 100，整理一下：
PB = PE × ROE% / 100
PE = 100 × PB / ROE%
代回 PR = PE / ROE%：
PR = PE / ROE% = (100 × PB / ROE%) / ROE%
👉 最终：PR = 100 × PB ÷ (ROE%)²
所以完整两种写法是：
公式一（用 PE）：
PR = PE / ROE%
公式二（用 PB）：
PR = 100 × PB / (ROE%)²
这俩完全等价，只是用的数据不同。
一般地：
利润（EPS）比较靠谱、没大波动时 → 用 PE 公式一
周期股、金融股、困境反转等 EPS 波动大 → 用 PB + ROE 的公式二

3. 标准计算流程 + 数字例子
假设一家公司情况如下：
股价 P = 20 元
EPS（TTM） = 2 元
ROE（TTM） = 20%
那么：
PE = 20 / 2 = 10 倍
ROE% = 20
BPS = EPS / ROE(小数) = 2 / 0.2 = 10 元
PB = 20 / 10 = 2 倍
用公式一算 PR：
PR = PE / ROE% = 10 / 20 = 0.5
用公式二算 PR：
PR = 100 × PB / (ROE%)²
= 100 × 2 / (20²)
= 200 / 400 = 0.5
两种口径给出的 PR 一致，说明逻辑没翻车。
这就是“原始市赚率”：拿 TTM 的 PE 和 ROE% 直接代入算出来的 PR。

二、各种“调整后”的市赚率：丁宁是怎么防止自己被骗的？
原始 PR 很好用，但现实世界很脏乱差：
今年 ROE 特别高/特别低（周期、疫情、一致性预期崩了…）
利润里掺了一次性收益、会计调整
公司加了很多杠杆，让 ROE 变“假高”
利率环境大幅变化（10 年国债 2% 跟 5%，合理估值肯定不同）
所以丁宁在框架里做了很多“修正版 PR”。大致是几条逻辑：

1）用“正常期 ROE”重算：归一化市赚率 PR_norm
目的：不要被当前这 12 个月的异常赚/亏骗了。
做法是：
把 TTM 的 ROE 换成一个“正常年份的 ROE 平均值”，再用公式二去算：
先定一个“正常期 ROE”：
比如过去 5 年中，剔除极端年份后的平均 ROE
或者困境反转，就用“危机前 3 年 ROE 的平均值”
用这个“正常期 ROE%”代入公式二：
PR_norm = 100 × PB ÷ (ROE_normal%)²
这相当于：
想象公司在“正常年份”的盈利能力（ROE_normal），
先算出“正常年份的 EPS_normal = BPS × ROE_normal(小数)”，
再算对应的 PE_normal，再除以 ROE_normal%，得到一个更稳定的 PR。
例子（接上面的公司）
刚才那家公司 TTM 数据是：
P=20，BPS=10，PB=2
TTM ROE = 20%，原始 PR = 0.5
假设你判断“20% ROE 太乐观，正常年份只有 15%”：
设 ROE_normal% = 15
用公式二算：
PR_norm = 100 × PB / (15²)
= 100 × 2 / 225 ≈ 0.89
含义：
原始 PR = 0.5，看起来像“半价”
但用“正常期 ROE”算，PR_norm ≈ 0.89，其实只是略微低估
所以“归一化 PR”是防止你被一两年超高 ROE 忽悠

2）利润质量/杠杆修正：修正市赚率 PR_adj = PR × N
有些公司 ROE 高，是因为：
杠杆开太猛
利润里一次性项目很多
经营现金流远低于净利润（会计利润水分大）
这种时候，丁宁的做法不是搞一套复杂公式，而是：
在心里给一个“质量系数 N（≥1）”。
修正市赚率：PR_adj = PR_norm × N
如果你觉得质量一般，N 可以取 1.1～1.3
如果觉得非常危险，N 可以 1.5 甚至 2（本质就是：按更“贵”的 PR 对待这家公司）
接上例子：
PR_norm ≈ 0.89
如果你认为利润质量一般，取 N = 1.2：
PR_adj = 0.89 × 1.2 ≈ 1.07
那就从“略低估”变成“略贵”，操作上可能就是：不买/持仓打折。

3）利率环境的调整：合理 PR 区间会随利率“漂移”
这块不是一个严格公式，而是对“合理 PR”的区间做环境修正：
粗暴理解：
无风险利率高（比如 10 年国债 4～5%）：
债券都能轻松躺赚，股市整体估值要打折
合理 PR 区间偏低，比如 0.7～0.9
无风险利率低（比如 10 年国债 2～3%）：
市场更愿意为权益资产付出溢价
合理 PR 区间可以上移，比如 1.1～1.4
你可以脑补一个“利率修正后的相对 PR”：
相对 PR ≈ 实际 PR ÷ 当下环境下的“合理中枢 PR*”
高利率环境下，如果 PR=0.9 可能只算“正常”
低利率环境下，同样 PR=0.9 就很便宜了
这里的重点是：不要把 PR=1 当成宇宙常数，而是会随利率整体调整。

4）困境反转专用：用“危机前正常期 ROE”算 PR
困境反转的典型操作是：
找到公司在“困境前”的几年的 ROE（比如 2015–2017 三年平均）
把那几年的平均 ROE 定为“正常期 ROE_normal”
用**当前的 PB + 正常期 ROE_normal%**算一个“正常期 PR”：
PR_recovery = 100 × PB_now ÷ (ROE_normal%)²
如果 PR_recovery 很低（比如 0.5 以下），说明：
假如公司能回到当年的盈利水平，现在价格是“大打折”
再叠加你对“回到正常”的概率判断，决定仓位
这个就是在丁宁框架里常说的：
用“正常期 ROE + 当前 PB”来定锚困境反转的机会。

5）用 PR 反推“合理价格”和“折扣价”——价格折扣表
这一步是把市赚率变成“能下单”的价格区间。
第一步：定一个“正常 ROE%”，记为 R_normal。
第二步：算对应的“PR=1 时的合理 PB”：
PB_fair = (R_normal²) / 100
推导来自：
当 PR=1 时，有 1 = 100 × PB_fair / (R_normal²)。
第三步：算合理价格 P_fair：
P_fair = PB_fair × BPS = BPS × (R_normal²) / 100
第四步：做一个“价格折扣表”：
6 折价：P_0.6 = 0.6 × P_fair
5 折价：P_0.5 = 0.5 × P_fair
4 折价：P_0.4 = 0.4 × P_fair
然后你有一个非常直观的操作表：
股价跌到 6 折 → 开始小仓
跌到 5 折 → 加到中仓
跌到 4 折 → 重仓
回到甚至高于合理价 → 慢慢减仓
接着用我们的例子：
之前算到：
BPS = 10 元
你认为“正常 ROE%” = 20
那么：
PB_fair = (20²) / 100 = 400 /100 = 4
P_fair = PB_fair × BPS = 4 × 10 = 40 元
当前股价 P_now = 20 元：
当前价格 = 20 / 40 = 0.5 × P_fair → 正好是5 折
和我们最开始算的 PR=0.5 完全对应
这就是丁宁用市赚率推导“折扣价”的套路：
先用 ROE + PB 定出合理价，再用价格折扣控制买入节奏。

6）金融/周期类的特化用法
这些算是“口径调整”，本质还是同一套公式：
强周期行业（钢铁、有色、化工…）
EPS 波动巨大，有时候 TTM EPS 甚至是亏损
更推荐：
先找一个“周期中枢 ROE_normal%”（比如 10 年中 3–5 个正常年份的平均）
用 PR_cycle = 100 × PB_now / (ROE_normal%)²
逻辑：别用“周期顶部”的暴利 ROE 来算便宜。
银行/保险等金融股
记账利润受拨备、利差等影响更大
常规操作：
用多年平均 ROE（10 年都行），得到一个 ROE_normal%
用 PB + ROE_normal 的公式算 PR
再结合监管红线（拨备覆盖率、资本充足率等）过滤雷

7）最终“多重修正版”的层级关系
把这些修正在脑子里排个层级：
原始市赚率：
PR_raw = PE_TTM / ROE_TTM%
归一化市赚率（正常期 ROE）：
PR_norm = 100 × PB_now / (ROE_normal%)²
质量修正市赚率（利润质量/杠杆）：
PR_adj = PR_norm × N_quality（N≥1）
环境解读（利率）：
在高/低利率环境下，对“PR_adj 是便宜还是贵”的判断阈值不同
困境反转专用：
ROE_normal 来自“危机前正常期”，PB 用当前的 → PR_recovery
操作层面的价格折扣表：
基于“PR=1 对应的 P_fair”，做 6 折/5 折/4 折的买入区间
你可以把它想成一套“滤镜”：
原始 PR → 换上“正常期 ROE”滤镜 → 再套“利润质量”滤镜 → 再用利率环境解读 → 最后落到“折扣价”的具体操作。

如果你愿意下一步，我们可以选一只你手上/关注的个股，从财报数据开始，一路算出：
PR_raw
PR_norm（自己定一个“正常 ROE”）
PR_adj（根据利润质量给个 N）
再推导出一个“合理价 + 6/5/4 折价表”，变成真正能用的买卖刻度尺。
