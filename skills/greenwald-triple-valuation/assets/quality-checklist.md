## Task 1：行业质地研究 — 质量核查

### 1.1 公司基础

- [ ] 主营业务及收入构成（按产品/地区/客户）已明确说明
- [ ] 最近年度总收入、营业利润、净利润三项数值均已载入
- [ ] 公司发展阶段（创业/成长/成熟/衰退）已判断并说明依据

### 1.2 行业格局

- [ ] 波特五力每项均已评分（高/中/低），并附1–2条具体依据
- [ ] 所在市场CR4/CR10集中度已查明（或给出合理估算区间）
- [ ] 行业生命周期（成长/成熟/衰退）已判断
- [ ] 主要竞争对手列表（≥3家）及各自市场份额已整理

### 1.3 护城河初判

- [ ] 已对成本优势、转换成本、监管许可、规模经济四类护城河逐一评估
- [ ] 每类评分有具体量化支撑（非纯定性描述）
- [ ] 初步护城河类型已给出（或明确判断为无护城河）

### 1.4 管理层质量

- [ ] 过去5年ΔMktCap/ΔRetainedEarnings已计算（巴菲特留存收益测试）
- [ ] 资本配置记录（并购回报、分红政策、CAPEX历史）已评价
- [ ] 关键业绩激励机制是否与股东利益对齐已评估

### 1.5 财务健康初步检查

- [ ] 近3年ROE、ROIC、FCF/净利润比值均已提取
- [ ] 流动比率和资产负债率已核查（与行业均值对比）
- [ ] 🔴 **KS-0初警**：应收账款增速 > 营收增速超过两年？若是，标注待Task 2深查

### 1.6 前置风险识别

- [ ] 致命风险（任一项可导致投资逻辑全毁）已单独列出
- [ ] 最少5条具体催化剂（正面+负面各≥2条）已识别
- [ ] 核心领先指标（Task 4追踪用）已提前记录

**Task 1 完成门槛**：1.6节无遗漏 + 所有项全部勾选 ✅

## Task 2：财务深度解构 — 质量核查

### 2.1 历史财务数据

- [ ] 损益表、资产负债表、现金流量表均覆盖≥5个完整财年
- [ ] 所有数据均标注来源（年报/Bloomberg/Choice/Wind）和报告日期
- [ ] GAAP与非GAAP指标的差异项已识别并说明

### 2.2 标准化处理

- [ ] 一次性损益项目（重组费用、减值损失、非常性收益）已逐一剔除
- [ ] SG&A"零增长部分"已按格林沃尔德标准加回（建议20–30%区间）
- [ ] R&D已按相同逻辑处理（若适用）
- [ ] 维护性CAPEX与成长性CAPEX已区分（至少两种估算方法交叉验证）

### 2.3 ROIC与FCF分析

- [ ] 5年ROIC序列已计算（选定一致的IC定义方法）
- [ ] WACC已按三个折现率场景（8%/9%/11%）准备
- [ ] ROIC-WACC超额收益时序已分析（趋势向好/稳定/恶化）
- [ ] FCF/净利润比值已计算，≥5年时序中的异常值已标注说明
- [ ] 🔴 **KS-1**：调整后NOPAT中，一次性项目占比 > 30%？若是 → 立即停止，标注致命

### 2.4 资产重置估算准备（AV前序）

- [ ] 各类固定资产重置成本基准已确定（新设备价格来源已记录）
- [ ] 商誉/无形资产处理规则已应用（✅/<20%保留  /  ❌/>20%打折50%）
- [ ] 🔴 **KS-2**：商誉/净资产 > 50%？若是 → 标注严重，强制50%减值

### 2.5 EPV输入准备

- [ ] 5年（或1周期）EBIT利润率中位数已记录
- [ ] 有效所得税率（实缴/法定差异）已分析
- [ ] 超额现金（= 总现金 - 收入×X%）已从资产中分离

**Task 2 完成门槛**：2.4、2.5节无缺项 + KS-1/KS-2均已处理 ✅

## Task 3：三重估值计算 — 质量核查

### 3.1 资产价值（AV）

- [ ] 流动资产调整（应收、存货）已完成，调整方法清晰
- [ ] 固定资产重置成本三场景（清算/重置/含无形）均已计算
- [ ] AV转换为每股价值（股权层面），已扣除有息债务

### 3.2 盈利能力价值（EPV）

- [ ] NOPAT四步推导链完整（无跳步）
- [ ] EPV = NOPAT / WACC已计算，三WACC场景均完成
- [ ] EPV已转为每股股权价值
- [ ] EPV/AV比值已计算并给出护城河初判结论

### 3.3 成长价值（GV）

- [ ] **GV前置检查**：EPV/AV ≥ 1.3 且定性护城河支持？若否 → GV = 0（记录原因）
- [ ] VCF（价值创造因子）= 再投资ROIC / WACC 已计算 → ≥1.0方可正值GV
- [ ] 护城河持续期已设定（≤10年上限，有论据）
- [ ] 三情景GV已计算（悲观/基准/乐观）
- [ ] 🔴 **KS-5**：行业明显衰退期 + 仍在计算正值GV？若是 → 立即调整为GV=0
- [ ] 🔴 **KS-3**：护城河期 > 10年或永续？若是 → 强制压回10年

### 3.4 估值全景汇总

- [ ] Football Field数据已完整（AV/EPV保守-基准-乐观/GV基准-乐观）
- [ ] **当前价格位置已标注**并计算安全边际（%）
- [ ] 🔴 **KS-4**：三估值之间差异 > 100%（如AV与EPV差双倍）？若是 → 需额外撰写差异解释段落，结论取保守端

### 3.5 关键假设

- [ ] 可证伪假设清单≥5条（每条含"若X发生则此假设失效"行动触发）
- [ ] 最重要的假设（对估值变化±10%以上）已在备忘录中重点标注

**Task 3 完成门槛**：3.4、3.5节无缺项 + 所有KS检查均已完成并记录 ✅
