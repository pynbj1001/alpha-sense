## 模板 C：成长价值（GV）评估

> **使用前提**：EPV/AV ≥ 1.3 且 Task 1定性支持护城河存在

### C1 自然增长参数

```
行业自然增长率参考（GDP/行业数据）     g_industry = [ %]
公司历史增长率（近5年中位）           g_historic  = [ %]
选定自然增长率                       g₁          = [ %]
每元新增收入所需新增IC（历史均值）     ΔIC/ΔRev   = [ ]x
维持自然增长所需年度再投资            re-invest₁  = 收入 × g₁ × ΔIC/ΔRev = [ ]
```

### C2 主动再投资参数

```
历史价值创造因子（VCF）验算：
  近5年平均年留存利润                            = [ ]
  近5年年均NOPAT增量                             = [ ]
  隐含再投资回报率                               = ΔNOPAT / 留存利润 = [X%]
  VCF = 再投资回报率 / WACC                     = [X%] / [9%] = [ ]x

判断：
  [ ] VCF > 1.0 → 有效再投资，可计入GV
  [ ] VCF ≈ 1.0 → 中性，建议以分红替代留存
  [ ] VCF < 1.0 → 价值毁灭，GV = 0
```

### C3 护城河持续期与有效增长率

```
护城河持续期（上限10年）  = [ ] 年
有效增长率 g_eff：
  悲观情景：g₁ × VCF保守值  = [ %]
  基准情景：g₁ × VCF中值    = [ %]
  乐观情景：g₁ × VCF乐观值  = [ %]
```

### C4 GV 计算

$$GV = EPV_{基准} \times \left(\frac{WACC}{WACC - g_{eff}} - 1\right)$$
