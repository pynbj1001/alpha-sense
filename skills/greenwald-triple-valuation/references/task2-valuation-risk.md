## 一、资产价值(AV)计算

### 调整方法
| 科目 | 账面值 | 调整方法 | 重置价值 |
|------|--------|---------|---------|
| 应收账款 | | -5%坏账 | |
| 存货 | | FIFO调整 | |
| 固定资产 | | 重置新价 | |
| 无形资产 | | R&D累积 | |
| 商誉 | | 审慎处理 | |
| 超额现金 | | 剔除(留1-2%收入) | |

### 计算公式
```
资产重置总值 = Σ调整后资产
企业AV = 资产重置总值 - 有息负债 - 运营负债
股权AV = 企业AV + 超额现金
AV每股 = 股权AV / 稀释股本
```

## 二、盈利能力价值(EPV)计算

### 正常化EBIT推导
```
报告EBIT
(+) 一次性收益加回
(+) SG&A加回(25%市场费用)
(+) R&D加回(25%前沿研发)
= 零增长调整后EBIT
(×) (1 - 有效税率)
= NOPAT
(-) 维护性CAPEX(D&A×30%)
= 净持续盈利
```

### 护城河判据
```
EPV/AV比值：
> 2.0 → 宽护城河 ✓
1.3-2.0 → 中等护城河
< 1.3 → 弱/无护城河
```

## 三、成长价值(GV)计算

### 前提检查
- [ ] EPV > AV（护城河存在）
- [ ] ROIC > WACC（有效再投资）
- [ ] 护城河期 ≤ 10年

### 计算公式
```
GV = EPV × (WACC/(WACC - g_eff) - 1)

g_eff = 行业自然增长率 × 再投资率 × VCF
VCF = ROIC / WACC（价值创造因子）
```

### 三情景GV
| 情景 | 护城河期 | g_eff | GV每股 | 含GV总价值 |
|------|---------|-------|--------|----------|
| 悲观 | 3年 | 4% | $ | $ |
| 基准 | 7年 | 8% | $ | $ |
| 乐观 | 10年 | 12% | $ | $ |
