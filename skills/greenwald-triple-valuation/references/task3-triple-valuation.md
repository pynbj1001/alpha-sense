# Task 3：三重估值计算 — 详细执行指引

## 目标

基于格林沃尔德框架，系统逐序计算三种估值：
**AV（资产价值）→ EPV（盈利能力价值）→ GV（成长价值）**

三种估值的关系：
- AV = 公司的"底线"安全垫（清算/重置视角）
- EPV = 当前竞争力的资本化（零增长永续价值）
- EPV - AV = 护城河量化值
- GV = 护城河保护下增长的额外回报（仅在护城河确认后计算）

**核心戒律**：只有在EPV明显高于AV（护城河有证据支撑）时，才计算并纳入GV。

---

## 执行流程

### 步骤 1：资产价值（AV）精算

**基础**：直接使用Task 2标签页4"资产重置估算"的输出数据。

**AV计算公式**：

$$AV(\text{股权}) = \sum \text{资产重置价值} - \text{有息债务} - \text{运营必要非付息负债} + \text{超额现金}$$

**格林沃尔德调整要点**：

| 资产类型 | 处理原则 | 格林沃尔德视角 |
|---------|---------|-------------|
| 营运现金 | 保留（收入1–2%） | 维持运营所必须 |
| 超额现金 | 先剔除后加回（股权层面） | 计入股权但不计入企业AV |
| 固定资产 | 按新设备重置价（非账面折旧净值） | 竞争视角：进入者需多少钱复制？ |
| 存货 | 调整为现行成本（FIFO） | 以当前重置成本而非历史成本 |
| 无形资产（可重建） | 重建成本法 | 纳入进入者需要的真实壁垒 |
| 商誉 | 谨慎处理（按规则分级） | 并购溢价不代表进入壁垒 |
| 递延税负债 | 通常剔除（非真实债务） | 会计负债非经济负债 |

**AV三个情景**：

| 情景 | 逻辑 | 使用场景 |
|------|------|---------|
| 悲观AV（清算价值） | 资产按变现折扣定价（流动资产8折，固定资产5折） | 行业衰退/公司困境 |
| 基准AV（重置成本） | 如上述逐科目调整 | 正常以上分析 |
| 乐观AV（含无形资产） | 将品牌/技术的重建成本全额计入 | 品牌价值显著的公司 |

**输出**：AV每股（悲观/基准/乐观）

### 步骤 2：盈利能力价值（EPV）计算

**基础**：使用Task 2标签页5"EPV输入准备"的输出数据。

**EPV公式（企业价值层面）**：

$$EPV_{EV} = \frac{\text{调整后税后持续盈利（净持续盈利）}}{WACC}$$

**EPV公式（股权价值层面）**：

$$EPV_{Equity} = EPV_{EV} + \text{超额现金} - \text{有息债务}$$

$$EPV_{\text{每股}} = \frac{EPV_{Equity}}{\text{稀释总股本}}$$

#### 关键调整的逻辑链

**1. 为什么要标准化利润（而非直接用当年净利润）？**

格林沃尔德认为EPV代表"永续零增长价值"，因此：
- 一次性损益扭曲了"永续"属性 → 剔除
- 周期高/低点的利润不代表中枢 → 用5年中位利润率正常化
- 异常高利润可能含增长性投入（会随增长停止而消失）→ 加回

**2. 为什么要加回SG&A / R&D的一部分？**

在零增长假设下，公司不需要进行市场拓展和前沿研发。
因此，用于"增长"的那部分费用会消失，变成盈利。
（注：维持现有市场份额所需的营销/维护性R&D必须保留。）

**3. 为什么要扣除维护性CAPEX（而非折旧）？**

折旧是会计概念；维护性CAPEX是真实的资本消耗。
对于资产大幅通胀的行业，折旧会低估实际更换成本。
（注：只扣除维持现有规模所必须的CAPEX，不扣增长CAPEX。）

**4. 资本成本（WACC）的选取**

格林沃尔德对CAPM持怀疑态度，推荐简洁稳定的折现率：
- 低风险稳定公司：8%
- 正常成熟公司（基准）：9%
- 周期性/不确定性较高：10%–12%
- 不应随市场波动频繁调整折现率

提供三个折现率的EPV区间（8%/9%/11%），以展示敏感性。

#### EPV的解读

- **EPV >> AV**：公司赚取了远超资本成本的回报 → 护城河存在（进入Task 4深度验证）
- **EPV ≈ AV**：竞争已将超额利润消耗殆尽 → 无护城河（GV = 0）
- **EPV < AV**：当前盈利低于资产正常回报 → 暂时困难/资源错配（深追原因）

---

### 步骤 3：护城河初判（EPV vs AV 比较分析）

| EPV/AV 比值 | 护城河判断 | 成长价值处理 |
|------------|---------|-----------|
| > 2.0 | 强护城河，超额利润明确 | 进入步骤4计算GV |
| 1.3 – 2.0 | 中等护城河，有超额利润迹象 | 谨慎计算GV，保守假设 |
| 1.0 – 1.3 | 弱护城河或周期高点 | GV = 0（不确定性太高） |
| < 1.0 | 无护城河或经营困难 | GV = 0，分析困境原因 |

**护城河初判结论必须与Task 1定性分析相互印证**：
- 若EPV/AV > 2 但Task 1未发现护城河来源 → 数据可能有误，重查
- 若Task 1有明确护城河描述但EPV/AV < 1.2 → 可能周期低点，或护城河正在侵蚀

### 步骤 4：成长价值（GV）评估

**前提条件**（任一不满足则GV = 0）：
- [ ] EPV/AV ≥ 1.3（有护城河迹象）
- [ ] Task 1已确认护城河来源（定性支撑）
- [ ] ROIC - WACC ≥ 5pp（历史验证有效再投资）

#### 4.1 自然增长价值

自然增长 = 行业需求增长 + 效率提升（不需要额外大规模资本投入）

**步骤**：
1. 确定自然增长率 $g_1$（参考行业增速、公司历史增速）
2. 计算维持该增长所需的年度再投资额：
   $\text{自然增长再投资} = \text{当期收入} \times g_1 \times \text{每元收入所需新增资产}$
3. 自然增长的净价值贡献（在EPV基础上）：

$$EPV_{\text{含自然增长}} = \frac{\text{净持续盈利}}{WACC - g_1}$$

（仅当 $g_1 < WACC$ 时此公式成立）

#### 4.2 主动再投资价值

公司在自然增长之外，主动将盈余再投资（并购/扩张/研发新品）所创造的价值：

**价值创造因子（Value Creation Factor, VCF）**：

$$VCF = \frac{ROIC_{\text{再投资}}}{WACC}$$

- VCF > 1.0：每1元再投资创造超过1元价值
- VCF ≈ 1.0：中性（建议以分红/回购代替再投资）
- VCF < 1.0：价值摧毁，禁止使用成长假设

**历史VCF推算方法**：
$$VCF_{\text{历史}} = \frac{\Delta E / \Delta \text{留存利润} \text{（近5年）}}{WACC}$$

（若历史上每留存1元产生的盈利增量突破WACC即为有效）

#### 4.3 成长价值汇总

$$GV = EPV \times \left(\frac{WACC}{WACC - g_{\text{eff}}} - 1\right)$$

其中 $g_{\text{eff}}$ 是在护城河期内的有效增长率（含自然增长 + 有效再投资带来的超额增长）

**护城河持续期上限**：
- 默认不超过10年
- 无特别深厚护城河不超过5年
- 超过10年需有极强证据，且需完整论证竞争动态

**强制三情景GV**：

| 情景 | 护城河期 | $g_{\text{eff}}$ | VCF | GV（每股） |
|------|---------|---------|-----|---------|
| 悲观 | 3–5年 | GDP-1% | 1.0 | |
| 基准 | 5–7年 | GDP+1% | 估算值 | |
| 乐观 | 7–10年 | GDP+3% | 估算值×1.2 | |

### 步骤 6：关键假设清单（可证伪格式）

**格式要求**：每条假设必须有可测量的反驳条件

```
假设1：正常化EBIT利润率 = X%（5年中位）
  可证伪条件：若下一财年利润率低于(X-5)%且非一次性原因，则需重新评估EPV

假设2：零增长调整加回比例 = SG&A 25%, R&D 25%
  可证伪条件：若公司主动宣布削减增长性投入但实际利润率未能按预测提升，
  则加回比例可能高估了增长性费用占比

假设3：WACC = 9%
  可证伪条件：若10年国债利率突破X%+300bp风险溢价，需重新测算WACC

假设4：护城河期 = Y年（仅GV适用）
  可证伪条件：若竞争对手在Z年内以同等成本达到公司规模的60%，则护城河期需缩短

假设5：价值创造因子 VCF = W
  可证伪条件：若下3年再投资ROIC降至WACC以下，则GV归零
```
