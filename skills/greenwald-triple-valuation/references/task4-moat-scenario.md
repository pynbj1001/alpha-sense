## Section 1：护城河深度验证

### 1.1 护城河宽度量化（必须使用至少2种方法）

#### 方法 A：ROIC - WACC 利差法（最直接）

| 指标 | 数值 | 结论 |
|------|------|------|
| 历史ROIC（近5年均值） | | |
| 估算WACC | | |
| ROIC - WACC 利差 | | |
| 利差趋势（扩张/稳定/收窄） | | |

**判断标准**：
- 利差 ≥ 10pp 且稳定5年+ → 宽护城河
- 利差 5–10pp 且稳定3年+ → 中等护城河
- 利差 2–5pp → 弱护城河
- 利差 < 2pp 或负值 → 护城河不存在

#### 方法 B：临界市占率法（格林沃尔德独特方法）

**核心逻辑**：新进入者进入市场后，需要达到什么规模（市占率）才能将ROIC提升到与在位者相当？
这一"临界市占率"越高，护城河越宽（因为新进入者需要垄断更大市场才能消除业绩劣势）。

**执行步骤**：
1. 确定行业的固定成本结构（固定成本占总成本的比例）
2. 估计规模效应曲线（单位成本随累计产量/市占率的下降斜率）
3. 计算新进入者达到在位者单位成本所需的最低市场份额
4. 与当前市场总容量对比

```
固定成本比例估算：
  在位者当前固定成本 ÷ 总成本 ≈ [X%]

新进入者临界市占率估算：
  若行业年销售规模为Y亿，
  进入者至少需要占据Z%才能达到规模平价

结论：
  临界市占率 [Z%] 占市场的意义：
  - Z% < 10%：低护城河（容易进入）
  - Z% = 10%–30%：中等护城河
  - Z% > 30%：宽护城河（进入成本极高）
```

## Section 2：Kill-Switch 红线检查（5项全部必检）

### KS-1：盈利可持续性

```
检查项：一次性收益占调整前EBIT的比例

计算：Task 2中一次性损益总额 ÷ 调整前EBIT
结果：[X%]

判断：
  [ ] X < 15%：通过 ✅
  [ ] X = 15%–30%：警告 ⚠️，标注风险但可继续
  [ ] X > 30%：🔴 致命触发

触发行动：标注"当前EPV盈利可持续性存疑，若剔除一次性项目则EPV降至[Y]，
          [ ] 高于市价 → 仍有安全边际（继续分析）
          [ ] 低于市价 → 立即降级为"观察"，禁止给出买入建议
```

### KS-2：商誉与无形资产风险

```
检查项：商誉 / 净资产比率

计算：最新商誉 ÷ 净资产账面值
结果：[X%]

判断：
  [ ] X < 20%：通过 ✅
  [ ] X = 20%–50%：⚠️ 附注，检查重大收购是否存在减值风险
  [ ] X > 50%：🟠 严重触发，AV需强制打折50%处理（已在Task 2执行）

商誉详情：各主要收购标的 + 商誉金额 + 当前运营状况
```

### KS-3：成长假设合理性

```
检查项：Task 3中GV使用的护城河持续期是否超过10年，
       或是否假设ROIC永久维持而不衰减

判断：
  [ ] 护城河期 ≤ 10年 且有衰减假设：通过 ✅
  [ ] 护城河期 > 10年 或无衰减假设：🟠 严重触发

触发行动：强制要求重新计算GV（缩短至5–7年 + 加入ROIC-WACC利差衰减系数）
         ROIC衰减模型：每年利差缩减 = (初始利差 - 0) / 护城河期
```

### KS-4：三重估值内部一致性

```
检查项：AV / EPV / (EPV + GV) 三者差异是否异常

计算：
  EPV / AV = [X]（护城河倍数）
  含GV估值 / AV = [Y]

判断：
  [ ] Y / AV < 3.0：合理范围 ✅
  [ ] Y / AV ≥ 3.0 且 < 5.0：⚠️ 需详细解释GV驱动因素
  [ ] Y / AV ≥ 5.0（即含GV估值 = 5倍以上AV）：🟡 警告触发

触发行动：Required — 详细说明差异来源，并将保守端（EPV）作为主要参考估值；
          乐观端（含GV）仅供参考
```

### KS-5：行业周期位置判断

```
检查项：当前行业是否处于明确衰退期，且分析仍使用成长估值给出买入

评估：
  行业当前阶段判断：导入期 / 成长期 / 成熟期 / 衰退期 / 颠覆期
  判断依据：[行业收入增速、市场渗透率、新进入者/退出情况]

判断：
  [ ] 行业处于成长期或成熟稳定期：通过 ✅
  [ ] 行业处于成熟期但增速放缓：⚠️ GV保守假设，成长期≤5年
  [ ] 行业处于明确衰退期：🔴 致命触发

触发行动：
  - 禁止使用成长估值（GV = 0）
  - 仅参考AV和调整后EPV（且EPV可能需下调）
  - 将"行业衰退"列为第一风险因素
```

