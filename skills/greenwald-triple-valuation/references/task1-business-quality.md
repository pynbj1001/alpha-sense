# Task 1：行业质地研究 — 详细执行指引

## 目标

为后续三重估值（AV / EPV / GV）提供全面的定性与定量基础。
本任务的核心输出必须回答三个关键问题：
1. 这家公司凭什么赚超额利润？（护城河来源）
2. 管理层是否有能力和意愿守护并扩大护城河？（资本配置）
3. 哪些风险可能在估值有效期内颠覆上述判断？（风险分层）

### Step 2：商业模式解析（500–800字）

按以下框架系统描述公司商业模式：

**2.1 价值主张**
- 公司向谁提供什么价值？（客户细分 + 核心价值）
- 为什么客户要选这家而不是竞争对手？

**2.2 收入模式**
- 收入结构（按产品线/地区拆分，需有历史数据）
- 一次性收入 vs 经常性收入（订阅/服务/耗材）的比例
- 收入驱动因子：量×价，哪个更重要？

**2.3 成本结构与运营杠杆**
- 固定成本 vs 变动成本比例
- 规模效应是否存在？（收入增长时利润率是否扩张）
- 运营杠杆系数（EBIT增速/收入增速）历史数据

**2.4 现金流特征**
- 商业模式是否产生"负营运资本"（先收款后付款）？
- FCF / 净利润 历史转化率（此处为定性判断，详细计算在Task 2）

### Step 4：护城河初步判断（600–800字）

格林沃尔德框架中，护城河的本质是"让在位者能持续赚取超额利润的机制"。
分四大类型逐一评估：

#### 4.1 成本优势型护城河

**核心问题**：公司能否以低于竞争对手的成本生产同等产品？

证据维度：
- [ ] 规模效应：市占率 vs 行业平均毛利率对比
- [ ] 独特资源获取：是否拥有稀缺低成本资产（矿藏/地段/专有技术）
- [ ] 学习曲线：单位成本是否随累计产量下降？
- [ ] 流程创新：专有制造流程/供应链优化

**结论**：此类护城河 [ ] 存在  [ ] 不存在  [ ] 不确定
**证据摘要**：[具体数据]
**持续性评估**：[可维持几年？是否容易被模仿？]

#### 4.2 客户粘性型护城河

**核心问题**：客户转换成本有多高？

证据维度：
- [ ] 高转换成本：数据迁移/习惯重建/重新学习成本
- [ ] 网络效应：用户越多，对单一用户价值越高（平台型）
- [ ] 品牌溢价：消费者愿意为品牌支付多少溢价？（同质量对比价格差异）
- [ ] 关系锁定：B2B场景的合同锁定/系统集成

**结论**：此类护城河 [ ] 存在  [ ] 不存在  [ ] 不确定
**证据摘要**：[具体数据，如客户留存率、NPS、续约率等]
**持续性评估**：[可维持几年？]

#### 4.3 法规与特许权型护城河

**核心问题**：政府/监管机构是否在保护公司的竞争地位？

证据维度：
- [ ] 牌照/许可：行业准入是否受限制？公司是否持有关键牌照？
- [ ] 专利组合：核心专利数量、剩余有效期、覆盖范围
- [ ] 监管标准：标准是否由公司参与制定？行业规范是否以公司能力为基准？
- [ ] 国资背景/战略资产：政策支持是否构成隐性保护？

**结论**：此类护城河 [ ] 存在  [ ] 不存在  [ ] 不确定
**证据摘要**：[具体内容]
**警告**：此类护城河有明确终止日期（专利到期/监管改变），需在Task 3估值中体现

#### 4.4 规模效应与高进入门槛型护城河

**核心问题**：新进入者需要多大的初始规模才能达到竞争力平价？

量化方法（格林沃尔德"临界市占率法"）：
- 测算在位者的边际成本曲线
- 推算新进入者在何种规模下才能达到同等单位成本
- 该"临界市占率"占市场总量的比例越高 → 护城河越宽

**结论**：此类护城河 [ ] 存在  [ ] 不存在  [ ] 不确定
**临界市占率估算**：[%，附计算逻辑]
**持续性评估**：[行业规模是否支持唯一主导者长期存在？]

**护城河初步总结**：
- 主要护城河类型：[列明]
- 综合评级（初步）：宽 / 中 / 窄 / 无
- 最大威胁：[最可能在5年内侵蚀护城河的因素]

### Step 7：风险因素分层清单（6–10项，不得减少）

每项风险必须包含：风险描述 / 触发条件 / 对估值影响 / 缓释措施

**分层规则**：
- 🔴 **致命级**：若触发，意味着护城河永久受损或盈利模式终结
- 🟠 **严重级**：若触发，EPV需下调15%以上，需重新评估建议
- 🟡 **一般级**：影响短期收益，不改变长期价值判断

**格式**：

```
🔴/🟠/🟡 [风险名称]
- 描述：[具体风险内容]
- 触发条件：[什么情况下会发生？概率评估？]
- 估值影响：[定量表述，如 EPV下调X%]
- 缓释措施：[公司已有/可能的应对手段]
```
