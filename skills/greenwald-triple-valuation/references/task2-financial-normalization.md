# Task 2：财务深度解构 — 详细执行指引

## 目标

将原始财务报表转化为三重估值所需的"干净数据"：
- **正常化盈利**：剔除噪音，得出可持续的EBIT/净利
- **资产重置基准**：为AV计算准备科目调整数据
- **ROIC/FCF解构**：为护城河判断和EPV计算提供量化依据

**核心原则**：格林沃尔德不信任预测，他信任当前可观察数据的精确处理。

## Excel模型结构（5个标签页）

### 标签页1：历史财务三表

**数据采集规范**：
- 最少5年，优选10年历史数据
- 按财年口径（非日历年）
- 中国公司以人民币为单位；海外公司保持原币后加汇率转换

| 销售费用 / 管理费用 / SG&A | 尽量细分 |

**资产负债表必提字段**（注意：固定资产需原值+累计折旧+净值三个数字）：

**现金流量表必提字段**：

CFO、其中折旧摊销加回、其中营运资本变动、CFI、其中CAPEX（取绝对值）、其中收购/处置支付、CFF、其中分红/回购/净借债。

**自由现金流**：FCF = CFO - 维护性CAPEX（非全部CAPEX，见标签页2）

### 标签页2：正常化利润表

**核心目标**：得出代表公司"在没有一次性干扰且零增长假设下"的可持续年度盈利。

#### 步骤 2.1：识别并剔除一次性损益

逐年检查非经常性项目，按以下规则处理：

| 项目类型 | 调整方向 | 注意点 |
|---------|---------|-------|
| 资产处置收益/损失 | 剔除 | |
| 资产减值损失 | 通常剔除 | 若频繁出现则视为经常性 |
| 政府补贴/税收返还 | 剔除（除非制度性可持续） | |
| 重组/并购相关费用 | 剔除 | |
| 诉讼赔偿 | 剔除 | |
| 汇率损益 | 特殊处理 | 主营汇率风险可部分保留 |

> **警告**：若一次性收益占调整前EBIT > 30%，触发Kill-Switch #1

#### 步骤 2.2：历史利润率正常化

- 计算每年调整后EBIT利润率
- 计算近5年中位数EBIT利润率
- 绘制利润率趋势图（扩张/收窄/稳定）
- 若当前利润率与历史中位偏差 > 30%，分析原因（护城河加深？还是周期高点？）

**正常化利润 = 当前收入 × 5年中位EBIT利润率**

#### 步骤 2.3：零增长调整（Greenwald核心操作）

在零增长假设下，部分SG&A和R&D属于"拓展性支出"可以节约：

| 费用类型 | 加回比例（建议范围） | 调整依据 |
|---------|----------------|---------|
| SG&A（市场拓展部分） | 10%–35% | 基准25%，按行业调整 |
| R&D（增长性研发部分） | 15%–35% | 基准25%，仅纯维护性R&D保留 |

**调整后EBIT = 原正常化EBIT + SG&A加回 + R&D加回**

> **必须**在假设清单中注明选择的加回比例及理由。

#### 步骤 2.4：维护性CAPEX 估算（三选一方法）

| 方法 | 公式 | 适用场景 |
|------|------|---------|
| 折旧近似法 | 维护CAPEX ≈ D&A × 80% | 资产密集型传统行业 |
| 收入比例法 | 维护CAPEX ≈ 历史CAPEX最低年份/收入 × 当前收入 | 通用方法 |
| 资产年龄法 | 根据累计折旧/原值估算更换周期 | 设备集中度高的行业 |

**正常化税后持续盈利 = (调整后EBIT × (1 - 有效税率)) - 维护性CAPEX**

### 标签页3：ROIC 与 FCF 解构

#### ROIC 计算

$$ROIC = \frac{NOPAT}{IC}$$

- **NOPAT** = 调整后EBIT × (1 - 有效税率)
- **IC（投入资本）** = 净营运资本 + 净固定资产 + 净无形资产（调整后）
- **净营运资本** = 流动资产 - 非生息流动负债（应付+预收+应付工资，不含短期借款）
- 商誉单独分析：计算含商誉/剔除商誉两个版本的ROIC

历史ROIC趋势表（≥5年）+ ROIC - WACC利差历史数据 + 3–5家同行ROIC对比

护城河量化信号：
- ROIC - WACC ≥ 5pp 且持续3年+ → 护城河硬判据
- ROIC趋势扩张 → 护城河加深，积极信号
- ROIC - WACC < 3pp 且收窄 → 护城河松动预警

#### FCF 转化率分析

计算每年 FCF/净利润（FCF = CFO - 维护性CAPEX）

| 质量评级 | FCF/净利润 | 行动 |
|---------|-----------|------|
| 高质量 | > 0.8（连续3年） | ✅ 正常进行 |
| 中等质量 | 0.5–0.8 | ⚠️ 分析原因 |
| 盈利质量警告 | < 0.5（连续2年+） | ❌ 触发Kill-Switch，Task 4需专项标注 |

### 标签页4：资产重置估算

逐科目调整，生成AV计算所需数据：

| 资产科目 | 账面值 | 调整方法 | 重置价值 | 调整说明 |
|---------|-------|---------|---------|---------|
| 货币资金（扣除超额现金） | | 保留收入1–2%作运营现金 | | |
| 应收账款 | | 扣坏账准备（历史/行业均值） | | |
| 存货 | | LIFO→FIFO调整（如适用） | | |
| 固定资产 | 账面净值 | 同等新设备当前市价（注明数据来源） | | |
| 在建工程 | | 按合同/预算完工价值 | | |
| 可重建无形资产 | | 重建成本法（R&D/营销累计） | | |
| 商誉 | | 见商誉处理规则 | | |
| 长期股权投资 | | 市值（上市）或净资产折扣 | | |
| **资产重置总值** | | | **合计** | |
| (-) 运营必要负债 | | 应付+预收+其他运营负债 | | |
| (-) 有息债务 | | 短期+长期借款+债券 | | |
| **股权层面AV** | | | **= 上差值** | |
| ÷ 总股本（稀释后） | | | **= AV每股** | |

**商誉处理规则**：
- 商誉/净资产 < 20%：保留
- 商誉/净资产 20%–50%：保留并注明风险
- 商誉/净资产 > 50%：**触发Kill-Switch #2**，强制打折50%
