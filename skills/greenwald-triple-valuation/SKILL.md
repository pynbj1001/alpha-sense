# Greenwald Triple Valuation Skill
## 格林沃尔德三重估值 — 买方机构级内在价值分析

基于布鲁斯·格林沃尔德《价值投资：从格雷厄姆到巴菲特（第2版）》核心框架，
通过严格的5任务工作流生成机构级内在价值研究报告。

**哲学基础**：先用确定性信息估算价值（资产 → 收益能力 → 成长），
护城河是成长价值的必要前提，安全边际是买入的充分条件。

## Task 2：财务深度解构

**目标**：提取历史财务数据，完成正常化处理，构建支撑三重估值所需的财务模型。

**前置条件**：⚠️ 执行前验证
- **必须**：可获取公司财务报表数据
  - 来源A：年报/10-K（SEC EDGAR 或交易所披露）
  - 来源B：可信财务数据供应商（Wind、Bloomberg、Choice等）
  - 来源C：用户提供的已整理历史财务数据

**输入验证**：
```
开始前确认：

选项A：直接提取财务数据
- [ ] 可访问年报或财务报表？
- [ ] 具备3–5年完整历史数据？

选项B：用户已提供整理好的数据
- [ ] 已收到历史财务文件？
- [ ] 包含三张报表（利润表/现金流/资产负债表）？
```

**执行流程**：
1. 确认数据来源
2. 从参考文件加载：`references/task2-financial-normalization.md`
3. 提取并正常化历史财务数据
4. 构建支撑估值的模型
5. 交付Excel模型

**产出**：财务解构模型 `.xlsx`（5标签页）
1. **历史财务三表**：最近5-10年原始数据（按季度/年度）
2. **正常化利润表**：剔除一次性项目，平滑周期波动，算出标准化EBIT/净利润
3. **ROIC & FCF 解构**：历史ROIC趋势（≥5年）、FCF/净利润转化率、再投资率
4. **资产重置估算工作表**：各科目调整明细，计算资产重置价值（AV初稿）
5. **EPV 输入准备**：标准化盈利基准、资本成本假设区间、维护性资本支出估算

**文件命名**：`[公司]_财务解构模型_[YYYY-MM-DD].xlsx`

**⚠️ 只交付此1份文件。**

**⚠️ 不得走捷径：**
- ✅ 提取全部三张报表历史数据（至少5年）
- ✅ 正常化利润需逐项说明调整理由
- ✅ ROIC计算必须使用调整后NOPAT和投入资本（非简单ROE代替）
- ✅ 资产重置工作表需对每个重要科目单独调整
- ✅ FCF/净利润比率必须在所有历史年度中计算
- ❌ 不得使用未调整的账面利润直接计算EPV
- ❌ 不得跳过资产重置估算工作表

**进入Task 3的验证标准**：
- [ ] Excel文件含全部5个标签页
- [ ] 正常化利润计算有明确调整说明
- [ ] 历史ROIC趋势图已生成（≥5年）
- [ ] AV初稿已填写
- [ ] EPV输入准备表已完成

**执行流程**：

3. 执行三重估值计算流程
4. 交付估值报告

**产出**：三重估值分析报告 + Excel估值标签页
- **文档**：`[公司]_三重估值分析_[YYYY-MM-DD].md`（3,000–4,500字）
  - 资产价值（AV）计算过程与结果

- ❌ 不得使用简单PE/PB代替AV/EPV计算

- ❌ 禁止给无护城河公司计算成长价值

**进入Task 4的验证标准**：

## Task 4：护城河验证与情景分析

**目标**：深度验证护城河可持续性，进行多维度情景与敏感性分析，
完成 Kill-Switch 红线检查。

**前置条件**：⚠️ **Task 3必须完成，否则立即停止**

**⚠️ 不完成Task 3直接执行Task 4将产生无效分析。**

**必须检查：**
```
开始前验证：
- [ ] Task 3已完成？（三重估值报告存在）
- [ ] 内在价值估值已确定？
- [ ] 护城河初判结论已有？
- [ ] 关键假设清单已填写？
```

**执行流程**：
1. 验证前置条件
2. 从参考文件加载：`references/task4-moat-scenario.md`
3. 执行护城河深度验证 + Kill-Switch检查 + 情景分析
4. 交付护城河与情景报告

**产出**：护城河与情景分析报告 `.md`（3,500–5,000字）
- **Section 1：护城河深度验证**
  - 护城河类型认证（四大类型 × 三维度评分）
  - 护城河宽度量化（临界市占率法 + ROIC-WACC利差法）
  - 护城河侵蚀风险矩阵（技术/监管/竞争/消费者偏好）
  - 护城河评级：宽/中/窄/无（附持续期假设）
- **Section 2：Kill-Switch 红线检查**（5项必检）
  - 盈利持续性判断（一次性收益占比）
  - 商誉/无形资产风险
  - 成长假设合理性（护城河期≤10年？）
  - 三重估值内部一致性校验
  - 行业周期位置判断
- **Section 3：情景与敏感性分析**
  - 三情景内在价值矩阵（悲观/基准/乐观）
  - 关键变量敏感性分析（WACC ±1%、增长率±2%、利润率±2%）
  - 护城河消退速度对估值的影响
  - 安全边际评估（vs 当前市场价格）
- **Section 4：领先指标监控看板**
  - 5–8个特异性定量触发指标
  - 每个指标附预警阈值和行动规则

**文件命名**：`[公司]_护城河与情景分析_[YYYY-MM-DD].md`

**⚠️ 只交付此1份文件。**

**⚠️ 不得走捷径：**
- ✅ Kill-Switch 5项全部检查，任意一项触发必须明确标注并说明影响
- ✅ 情景分析必须有数字区间（不接受"较高/较低"等定性描述）
- ✅ 护城河宽度量化必须包含 ROIC-WACC 利差历史数据
- ✅ 安全边际计算必须基于保守情景（EPV或AV），而非乐观情景
- ❌ Kill-Switch 检查不得省略任何一项
- ❌ 情景分析不得使用"最好情况"和"最差情况"而不提供数字

- [ ] Kill-Switch 5项检查全部完成，结论明确

- [ ] 安全边际百分比已计算（vs当前价格）

## 三重估值框架核心速查

### 资产价值（AV）— 底线与安全垫

$$AV = \sum \text{调整后资产重置价值} - \text{有息负债} - \text{运营必要负债}$$

| 科目 | 调整方法 |
|------|---------|
| 应收账款 | 扣除坏账准备（2–5%） |
| 存货 | LIFO→FIFO 调整 + 当前成本 |
| 固定资产 | 按重置新价（非折旧账面值） |
| 无形资产 | 重建成本法（R&D累积 + 品牌投入） |
| 超额现金 | 剔除（仅保留运营所需现金，≈年收入1–2%） |
| 商誉 | 审慎处理，商誉/净资产>50%需打折 |

### 盈利能力价值（EPV）— 当前竞争力的资本化

$$EPV = \frac{\text{调整后持续税后盈利}}{R} = \frac{\text{标准化EBIT} \times (1-t) - \text{维护性资本支出}}{WACC}$$

**零增长调整**：SG&A 和 R&D 加回25%（参考标准，按行业调整）

**护城河判断**：
- `EPV/AV > 2.0`：强护城河确认
- `EPV/AV ∈ [1.3, 2.0]`：中等护城河
- `EPV/AV ∈ [1.0, 1.3]`：弱护城河或周期高点
- `EPV/AV < 1.0`：经营不善 / 资源错配

### 成长价值（GV）— 护城河赋予的增长期权

**前提**：仅在 EPV > AV 且护城河可证明时计算

$$GV = EPV \times \left(\frac{R}{R - g_{\text{eff}}} - 1\right)$$

其中 $g_{\text{eff}}$ 仅在 ROIC > WACC 的护城河期内有效（默认≤10年）

**价值创造因子**：
- `>1.0`：有效再投资，增长创造价值
- `≈1.0`：中性（以分红/回购为主）
- `<1.0`：价值毁灭，拒绝成长假设

### 量化阈值速查

| 指标 | 阈值 | 含义 |
|------|------|------|
| 市价/AV | < 0.7 | 深度低估，格雷厄姆底线 |
| EPV/AV | > 2.0 | 强护城河确认 |
| FCF/净利润 | > 0.8 | 高质量盈利 |
| FCF/净利润 | < 0.5（连续2季） | 盈利质量警告 |
| ROIC - WACC | ≥ 5pp（持续3年+） | 护城河硬判据 |
| 安全边际（EPV） | ≥ 30% | 买入门槛（无增长情景） |
| 安全边际（含GV） | ≥ 40% | 买入门槛（含成长情景） |
| 护城河持续期 | ≤ 10年 | 成长假设上限 |

## Kill-Switch 红线（强制检查）

| # | 触发条件 | 严重度 | 行动 |
|---|---------|-------|------|
| 1 | 一次性收益占调整前EBIT > 30% 且剔除后 EPV < 市价 | 致命 | 停止分析，标注"盈利不可持续" |
| 2 | 商誉/净资产 > 50% 或无形资产无法验证重置成本 | 严重 | 资产价值强制打折50%作为保守下限 |
| 3 | 成长假设护城河期 > 10年 或 ROIC 永久不衰减假设 | 严重 | 强制缩短至5–7年并加入衰减系数 |
| 4 | 三重估值结果差异 > 100%（如 AV=10，含GV估值=30+） | 警告 | 必须详细解释差异来源，取保守端为主参考 |
| 5 | 行业处于明确衰退期 + 分析仍使用成长估值给出买入 | 致命 | 禁止使用成长估值，仅参考 AV 和调整后EPV |

## 二阶思考强制检查

每次完成三重估值后，必须追问：

1. **"如果 EPV >> AV，然后呢？"** — 护城河是否正在被侵蚀？高 ROIC 是否正在吸引模仿者？
2. **"市场共识是什么？"** — 如果所有分析师都认同护城河，成长溢价还是"白送的"吗？
3. **"利润改善是渐进的还是有拐点？"** — 规模效应是否可能在某个阈值后突然饱和？
4. **"反身性？"** — 高估值允许低成本融资并购，可能自我实现护城河扩大；反之亦然
5. **"零增长假设下，价格仍有安全边际吗？"** — 格林沃尔德框架最核心的测试
