## Phase 0：巴菲特一元测试（硬性门控，优先执行）

```bash

python tools/retained_earnings_check.py --ticker [代码] --years 10 --save
```

| REM | 对后续估值的强制影响 |
|:---:|---|
| **< 1** | 🚨 **报告顶部打红色警告**；EPV × 70%；成长价值 = 0；所有成长论点需更高举证标准 |
| **1 ~ 2** | ⚠️ EPV × 90%；成长价值需额外举证（至少1项可证伪证据）|
| **> 2** | ✅ 正常权重，EPV × 100%，进入完整三重估值 |

**REM < 1 时，强制在报告顶部插入：**

```

   REM = [X]x（< 1）

   EPV 已按 70% 折价计算；成长价值不纳入估值。
   建议关注：管理层是否应停止低效留存，转为分红/回购。
```

**必须获取并标注来源+日期：**

| 指标 | 用途 | 来源 |
|---|---|---|
| EPS（TTM / 下一年度预测）| EPV 基础数字 | yfinance / MCP |
| EBIT（调整后）| EPV 基础数字 | 财报计算 |
| WACC | 折现率 | CAPM（见计算方法）|
| 净资产（重置价值）| 资产价值 | 财报 |
| PE（当前 / 历史中位）| PR 估值 | yfinance |
| ROE（近5年均值）| PR 公式 + REM 输入 | 财报 |
| 股息率 | g* 计算 | yfinance |
| Net Debt（净负债/净现金）| EV → 股权价值桥接 | 财报 |
| 稀释股本 | 每股价值计算 | yfinance |

**WACC 计算方法（精确，不得用经验拍值）：**
```
Risk-Free Rate = 当前10年期国债收益率（美股用10Y UST；A股用10Y CGT）
Beta = 5年月度股价 vs 市场指数（yfinance 可直接获取）
Equity Risk Premium = 5.0~6.0%（美股）/ 6.0~7.5%（A股）

Cost of Equity = Risk-Free Rate + Beta × ERP
Cost of Debt = 有息负债利息支出 / 平均有息负债余额（税后）
Equity Weight = 总市值 / (总市值 + Net Debt)
Debt Weight = Net Debt / (总市值 + Net Debt)

WACC = Cost of Equity × Equity Weight + Cost of Debt (after-tax) × Debt Weight

注意：若 Net Debt < 0（净现金），Debt Weight 为负，WACC 略低于 Cost of Equity
```

## Phase 2：格林沃尔德三重估值

### ① 资产价值（Asset Value）— 清算视角

```
资产价值 = 流动资产（按市值调整）+ 有形固定资产（重置成本）

调整规则：
- 应收账款：账面值 × 90%（坏账折扣）
- 存货：账面值 × 70~80%（变现折扣）
- 固定资产：重置成本（通常为账面净值 × 1.0~1.5x，与设备更新成本对比）
- 商誉/无形资产：若无竞争优势证明，清零
- 负债：按面值全额扣减
```

### ② 盈利能力价值（EPV）— 持续经营视角

```
EPV 计算（三步法）：

Step 1：确定 Normalized EBIT
  = 近5年EBIT均值（剔除非经常损益和周期性峰谷）
  调整项：+研发摊销（若资本化）/ ±重组费用 / ±资产减值

Step 2：计算 NOPAT
  NOPAT = Normalized EBIT × (1 - Tax Rate)
  Tax Rate：使用实际有效税率（近5年均值，而非法定税率）

Step 3：EPV = NOPAT / WACC

Step 4：REM 折价调整
  Final EPV = NOPAT / WACC × REM折价系数（见Phase 0表格）

Step 5：EV → 股权价值
  股权价值 = Final EPV - Net Debt
  每股 EPV = 股权价值 / 稀释股本
```

### ③ 成长价值（Growth Value）— 仅当有护城河时计入

**前置条件（两项必须同时满足，否则成长价值 = 0）：**
- 条件A：ROIC > WACC（超额回报存在，当前期）
- 条件B：存在可持续竞争优势（护城河评级 ≥ "窄护城河"）
- 条件C（强制）：REM > 1

```
成长价值 = EPV × 成长溢价倍数

成长溢价倍数估算（Ward方法）：
  设增长率 g，超额回报期 N 年（通常3~10年）

  每年超额价值创造 = (ROIC - WACC) / WACC × 新增投资
  成长溢价 = EPV × g × (ROIC - WACC) / (WACC - g) × 折现年限系数

  简化版（适用L2/L3）：
  成长价值 = EPV × (g / WACC) × (ROIC/WACC - 1) × 折现年限系数
```
