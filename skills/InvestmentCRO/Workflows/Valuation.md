# Valuation Workflow — 估值工作流

适用：`@估值 [公司]`

---

## 步骤

1. **拉取数据**（标注来源+日期）：当前股价、PE、PB、PS、EV-EBITDA、ROE、FCF、g、股息率

2. **三重估值（格林沃尔德）**
   - 资产价值：重置成本估算
   - 盈利能力价值（EPV）：adjusted_earnings / WACC
   - 成长价值：仅当存在可持续竞争优势时计入

3. **市赚率（PR）交叉验证**
   - PR = PE / ROE×100
   - PR < 1 = 便宜；PR > 2 = 贵
   - 计算盈亏平衡增长率 g* = 1/PE - 股息率

4. **TSR分解**
   - 预期总回报 = 盈利增速g + 股息率d + 估值变动ΔPE
   - 与无风险利率比较

5. **安全边际评估**
   - 当前价格 vs EPV
   - 若安全边际 < 20%（无护城河企业）→ Kill-Switch警告

6. **散户乙十年回本测算**（仅适用稳定盈利企业）
   - 年化自由现金流 / 总市值 = 回报率
   - 10年累计回本比例

7. **输出**：估值区间表 + 关键假设 + 敏感性分析（关键变量±10%）
   - 存入 `10-研究报告输出/`
