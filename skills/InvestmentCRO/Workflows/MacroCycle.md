# MacroCycle Workflow — 宏观周期工作流 v5.0

适用：`@宏观`、`@周期`

---

## ⚠️ CRITICAL CONSTRAINTS

❌ **NEVER** 用记忆中的历史宏观数据（必须实时拉取，标注日期）  
❌ **NEVER** 给出模糊的资产配置建议（必须给出具体比例，如「股票：40%」）  
❌ **NEVER** 省略四情景概率（概率之和必须=100%）  
❌ **NEVER** 将康波预判用于短期个股操作（康波是超长周期，仅指导大类资产方向）

**最低交付标准：**
- ✅ 核心宏观指标快照（含数据日期和来源）
- ✅ 三周期定位（康波/朱格拉/基钦）
- ✅ 象限判断（PMI×CPI 四象限）
- ✅ 大类资产配置建议（股/债/商品/现金，具体比例）
- ✅ 四情景及概率（之和=100%）

```
文件命名：10-研究报告输出/[YYYY-MM-DD]-宏观周期.md
```

---

## 步骤

1. **获取实时宏观数据**（标注来源+日期）
   ```python
   import akshare as ak
   # M2、CPI、PMI、社融
   df = ak.macro_china_money_supply()
   ```
   - GDP增速 / CPI / PPI / PMI
   - M2增速 / 社融增速 / M2-M1剪刀差
   - 10年国债收益率 / 期限利差（10Y-2Y）
   - LPR 1Y/5Y / 汇率

2. **周金涛康波定位**
   - 当前处于康波周期（54年）的哪个阶段？
   - 当前处于朱格拉周期（10年）的哪个阶段？
   - 当前处于基钦周期（3-4年）的哪个阶段？

3. **周期象限定位**（基于PMI+CPI）
   - 扩张期（PMI↑，CPI↑）→ 偏股，减债
   - 滞胀期（PMI↓，CPI↑）→ 减股，加商品
   - 衰退期（PMI↓，CPI↓）→ 偏债，减股
   - 复苏期（PMI↑，CPI↓）→ 偏股，减债

4. **资产配置建议**
   - 股票：配置比例建议
   - 债券：久期建议
   - 商品：配置逻辑
   - 现金：防御比例

5. **情景树**（含概率）
   - 软着陆 / 硬着陆 / 再通胀 / 滞胀

6. **存入 `10-研究报告输出/`**
