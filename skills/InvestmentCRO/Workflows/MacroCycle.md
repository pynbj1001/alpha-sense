# MacroCycle Workflow — 宏观周期工作流

适用：`@宏观`、`@周期`

---

## 步骤

1. **获取实时宏观数据**（标注来源+日期）
   ```python
   import akshare as ak
   # M2、CPI、PMI、社融
   df = ak.macro_china_money_supply()
   ```
   - GDP增速 / CPI / PPI / PMI
   - M2增速 / 社融增速 / M2-M1剪刀差
   - 10年国债收益率 / 期限利差（10Y-2Y）
   - LPR 1Y/5Y / 汇率

2. **周金涛康波定位**
   - 当前处于康波周期（54年）的哪个阶段？
   - 当前处于朱格拉周期（10年）的哪个阶段？
   - 当前处于基钦周期（3-4年）的哪个阶段？

3. **周期象限定位**（基于PMI+CPI）
   - 扩张期（PMI↑，CPI↑）→ 偏股，减债
   - 滞胀期（PMI↓，CPI↑）→ 减股，加商品
   - 衰退期（PMI↓，CPI↓）→ 偏债，减股
   - 复苏期（PMI↑，CPI↓）→ 偏股，减债

4. **资产配置建议**
   - 股票：配置比例建议
   - 债券：久期建议
   - 商品：配置逻辑
   - 现金：防御比例

5. **情景树**（含概率）
   - 软着陆 / 硬着陆 / 再通胀 / 滞胀

6. **存入 `10-研究报告输出/`**
