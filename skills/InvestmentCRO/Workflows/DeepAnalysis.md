# DeepAnalysis Workflow — 深度分析工作流

适用：`@分析`、`@行业`、`@L4`、`@L5`、`@L6`、`@沙盘`、`@对比`

---

## 预检清单（每次必做）

1. 能力圈检查 — 商业模式能否30秒向外行解释清楚？
2. 数据可得性 — 是否有公开财务数据？
3. 生命周期 — 初创/成长/成熟/衰退？→ 决定框架权重
4. 周期位置 — 行业周期+宏观周期当前位置？
5. 市场情绪 — 市场共识是什么？过度乐观/悲观？
6. 历史报告 — 检查 `10-研究报告输出/` 有无既往分析
7. 时间框架 — 用户关心短期(<1年)/中期(1-3年)/长期(3+年)？

---

## 执行步骤

### Step 1：数据先行
用 Python 实时获取，标注来源+日期：
- 估值：PE / PB / PS / EV-EBITDA / 市赚率PR
- 质量：ROE / ROIC / MROIC / 毛利率 / 净利率 / FCF
- 成长：营收增速 / 利润增速 / g（可持续增长率）
- 资本：负债率 / 有息负债 / Net Debt/EBITDA
- 股东回报：股息率 / 回购率

### Step 2：框架匹配（按情景选择）
- S1个股：六层金字塔 + 三重估值 + 护城河 + Q-G-P-R
- S2行业：三重共振 + 产业拼图 + 波特五力 + 渗透率天花板
- S5成长：10X Alpha + 技术革命 + BG韧性 + 贝叶斯拐点

### Step 3：多框架并行分析（三维交叉）

**A. 价值维度（"值多少钱"）**
- 格林沃尔德三重估值（资产→EPV→成长价值）
- 卡拉曼安全边际评估
- 市赚率PR / 散户乙十年回本

**B. 成长维度（"能长多大"）**
- 贝叶斯拐点概率更新
- 费雪15问成长质量检验
- 戴维斯双击可能性

**C. 宏观/周期维度（"时机对不对"）**
- 康波周期定位
- 三重共振（周期×客户×产品）
- 钟摆指数市场温度

每个框架输出：结论(1句) + 置信度(%) + 关键假设(≤3条可证伪) + 二阶推演

### Step 4：前瞻性分析
- 领先指标映射（每个论点1-3个领先指标）
- 催化剂时间表
- 情景分析：牛/基/熊/黑天鹅（含概率、目标价、触发信号）
- 二阶效应推演

### Step 5：框架冲突仲裁
- 估值便宜+成长差 → 价值陷阱风险，深入验证
- 估值贵+成长好 → GARP机会，验证g持续性
- 所有框架一致看好 → 检查是否全面共识
- 最终裁决：Q-G-P-R打分卡综合

### Step 6：Kill-Switch检查（见 SKILL.md）

### Step 7：概率化综合判断
- 贝叶斯加权汇总
- TSR分解：g + d + ΔPE
- 盈亏平衡增长率g*：当前PE隐含的市场预期

### Step 8：输出报告
存入 `10-研究报告输出/[YYYY-MM-DD]-[类型]-[标的]-[框架].md`

## L5/L6 特殊模式

**L5 买方决策（`@L5` / "CIO备忘录"）**
- UE模型：单店/单客经济学推演
- 敏感性测试：关键变量变化1%的影响
- 非共识数据：市场忽略的隐形资产或风险
- 输出：CIO备忘录，说人话，聚焦Key Variable与赔率

**L6 沙盘推演（`@L6` / `@沙盘` / `@红队`）**
前置：须先完成L5备忘录
- 红蓝军对抗：多空双方各构建完整论证，多轮交锋
- 战局推演：多参与方博弈模拟 + 决策树
- 终局图景：T+5/T+10年行业终态概率分布
- 末日协议：黑天鹅压力测试 + 对冲方案
- 执行兵法：建仓/持仓/退出战术 + Tripwire触发系统
