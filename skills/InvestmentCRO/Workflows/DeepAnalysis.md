## Phase 3：多框架并行分析（三维交叉）

**每个框架输出格式（严格遵守）：**
> 结论（1句话，含明确方向）| 置信度：XX% | 关键假设：①... ②... ③... | 二阶推演：...

### 维度A：价值（"值多少钱"）

**① 格林沃尔德三重估值**
```
资产价值（Asset Value）= 重置成本
  → 固定资产重置 + 无形资产账面值（若无竞争优势则归零）

盈利能力价值（EPV）= 调整后息税前利润 / WACC
  → 调整项：剔除非经常损益、研发资本化、周期利润均值化
  → REM < 1 时：EPV × 70%（折价）
  → REM 1~2 时：EPV × 90%

成长价值（Growth Value）
  → 仅当 (ROIC > WACC) AND (存在可持续竞争优势) 时计入
  → REM < 1 时：成长价值一律清零
```

**② 市赚率（PR）交叉验证**
```
PR = PE / (ROE × 100)
PR < 1  → 价值洼地
PR 1~2  → 合理区间
PR > 2  → 偏贵，需成长支撑

盈亏平衡增长率 g* = (1/PE) - 股息率
含义：当前价格隐含市场预期增速
若实际增速 >> g* → 被低估；若实际增速 << g* → 被高估
```

**③ 资本配置质量（巴菲特一元测试，五项指标）**
- 详细计算见 `RetainedEarningsCheck.md`
- 此处汇报综合评分 + 五项指标等级

### 维度B：成长（"能长多大"）

**④ 贝叶斯拐点概率更新**
```
先验概率（Base Rate）：同类公司历史成功率
似然比（Likelihood Ratio）：
  - 正面证据：+ 更新概率
  - 负面证据：- 降低概率
后验概率 = 先验 × 似然比（归一化）

拐点类型：
  - 行业渗透率拐点（S曲线加速段）
  - 盈利拐点（亏损转盈利的临界点）
  - 估值拐点（市场重新定价触发）
```

**⑤ 费雪15问成长质量检验（精简版，必查5项）**
- Q1：产品/服务市场规模是否足够大？（TAM > 市值5x）
- Q5：利润率是否可持续提升？（毛利率趋势向上）
- Q7：劳动关系是否良好？（员工满意度/离职率）
- Q9：管理层深度是否充足？（CFO/CTO任期≥3年）
- Q15：若价格过高，管理层是否仍诚实对待投资者？

**⑥ 戴维斯双击可能性**
```
条件：① 盈利增速加速 AND ② 估值倍数扩张空间存在
概率评级：高/中/低
典型路径：困境反转期 → EPS低基数 + PE从底部扩张
```

### 维度C：宏观/周期（"时机对不对"）

**⑦ 康波周期定位**
- 当前位于康波第几波（上行/繁荣/衰退/萧条）
- 对应行业影响（资源/制造/金融/消费各有不同）

**⑧ 三重共振检验**
```
共振维度：行业周期 × 公司竞争力 × 估值安全边际
三维共振（+/+/+）→ 强买入信号（置信度 ≥ 70%）
二维共振（+/+/-）→ 谨慎买入，等待估值改善
单维共振          → 观察，不行动
```

**⑨ 市场情绪温度计**
```
钟摆指数 = 当前PE/历史中位PE
> 1.3 → 过热（贪婪区）
0.8~1.3 → 合理
< 0.8 → 低温（恐惧区，价值买入机会）
```

❌ 不得省略触发信号（每个情景必须有具体可观测信号）

### TSR 分解（定量化）

$$TSR = g_{盈利增速} + d_{股息率} + \Delta PE_{估值变动}$$

```
示例：
当前PE：15x  目标PE：18x（PE扩张）
EPS增速：12%  股息率：2%
预期TSR = 12% + 2% + (18/15 - 1) = 12% + 2% + 20% = 34%（3年期）
年化 ≈ 10.3%
```

## Phase 5：框架冲突仲裁与 Kill-Switch

### 冲突仲裁规则

| 组合 | 判断 | 行动 |
|---|---|---|
| 估值便宜 + 成长差 | 价值陷阱风险 | 深挖 REM 和 ROIC 趋势，检查是否护城河侵蚀 |
| 估值贵 + 成长好 | GARP 机会 | 验证 g 可持续性（费雪5问 + 贝叶斯拐点）|
| 所有框架一致看好 | 全面共识风险 | **Kill-Switch 触发** — 市场已充分定价，寻找非共识点 |
| 估值合理 + 周期顶部 | 时机风险 | 分批建仓，加大对催化剂的监测 |

### Kill-Switch（任一触发 → 立即降为"观察"）

```
Kill-Switch 检查清单
[ ] 核心数据无法获取或不一致？
[ ] 商业模式无法30秒解释？
[ ] 管理层红旗（造假迹象/频繁离职/不寻常关联交易）？
[ ] 有息负债/EBITDA > 4x？
[ ] 安全边际 < 20%（无护城河企业）？
[ ] 单一客户/行业来源 > 50%？
[ ] 全面市场共识（无非共识观点可发掘）？

任一为 YES → 结论降级为"观察"，在报告中显著标注原因
```

## Phase 6：概率化综合判断

### Q-G-P-R 打分卡（综合裁决，满分100）

| 维度 | 权重 | 得分 | 关键依据（1句话）|
|---|:---:|:---:|---|
| **Q 质量**（REM + ROE + ROIC + FCF转化率）| 30% | /30 | |
| **G 成长**（营收CAGR + 可持续增长率 + TAM空间）| 30% | /30 | |
| **P 价格**（安全边际 + PR + 历史估值分位）| 25% | /25 | |
| **R 风险**（负债率 + Kill-Switch + 催化剂确定性）| 15% | /15 | |
| **综合得分** | 100% | /100 | |

**评级标准：**
- 80-100：🌟 强研究候选（值得深入+建仓）
- 65-79：✅ 关注（继续跟踪；等待价格或催化剂）
- 50-64：⚠️ 持有观察（现有仓位不扩大）
- < 50：❌ 规避（存在重大风险或无安全边际）

### L5 买方决策（`@L5` / "CIO备忘录"）

**⚠️ CRITICAL：L5 必须包含以下三项，缺一不可**

**① UE 模型（单位经济学）**
```
单店/单客经济学推演（选择适用的业务单元）：

CAC（获客成本）= 销售费用 / 新增客户数
LTV（客户生命周期价值）= ARPU × 毛利率 × 平均留存年限
LTV/CAC 比率（> 3x 为健康）

单店模型：
  投资成本：XX 万
  成熟期年 EBITDA：XX 万
  回本周期：XX 年
  系统级 IRR：XX%
```

| 关键变量 | 基准值 | +1% 时目标价 | -1% 时目标价 | 敏感度级别 |
|---|---|---|---|---|
| 营收增速（g）| XX% | ¥XX | ¥XX | 高/中/低 |
| EBIT 利润率 | XX% | ¥XX | ¥XX | 高/中/低 |

**③ 非共识数据（市场忽略的隐形资产或风险）**
