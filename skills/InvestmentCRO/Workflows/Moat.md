# Moat Workflow — 护城河工作流 v5.0

适用：`@护城河 [公司]`

---

## ⚠️ CRITICAL CONSTRAINTS

❌ **NEVER** 在 ROIC 趋势数据缺失时给出高护城河评级（无数据 = 无护城河假设）  
❌ **NEVER** 省略可证伪条件（每项护城河判断必须有「如果X发生则判断错误」的条件）  
❌ **NEVER** 仅用定性描述替代 ROIC 数字（必须有近5年 ROIC 数据支撑）  
❌ **NEVER** 跳过 REM 检验（REM < 1 → 护城河评级强制降一档）

**最低交付标准：**
- ✅ ROIC 趋势表（近5年，标注来源）
- ✅ 巴菲特9问（每问有具体数据支撑，不得泛泛作答）
- ✅ 护城河评级 + 置信度 + 可证伪条件
- ✅ REM 对评级的影响说明

```
文件命名：10-研究报告输出/[YYYY-MM-DD]-护城河-[标的].md
```

## Phase 1：数据获取（先于定性分析）

```python
import yfinance as yf
ticker = "[代码]"
stock = yf.Ticker(ticker)
# 计算 ROIC（近5年）= NOPAT / 投入资本
# python tools/retained_earnings_check.py --ticker [代码]
```

获取：ROIC（近5年逐年）/ 毛利率趋势 / 市占率变化 / 定价能力证据 / REM

**REM 对护城河评级的影响：**
- REM < 1 → 护城河评级**强制降一档**（宽→窄，窄→过渡，过渡→无）
  说明：管理层无法有效再投资，即便存在竞争优势也无法持续转化为股东价值

---

## Phase 2：护城河分析

**数据准备**：ROIC趋势（近5年）、毛利率趋势、市占率变化、定价能力证据

2. **巴菲特护城河9问**
   - 是否拥有无形资产（品牌/专利/许可）？持续性如何？
   - 是否有转换成本（客户粘性）？量化流失率
   - 是否具备网络效应？用户增长→价值增长的飞轮
   - 是否有成本优势（规模/流程/地理）？
   - 是否有高效规模（有限市场的自然垄断）？
   - 护城河是在拓宽还是收窄？（关键趋势问题）
   - 管理层是否在合理分配资本？
   - 主要竞争威胁是谁？威胁程度？
   - 10年后这家公司的护城河还在吗？

3. **波特五力验证**
   - 新进入者威胁 / 替代品威胁 / 买方议价力 / 供方议价力 / 行业竞争强度

4. **ROIC分析**
   - ROIC > WACC = 创造价值
   - MROIC（边际ROIC）趋势：新增投资回报率是否上升？

5. **护城河评级**
   - 宽护城河 / 窄护城河 / 无护城河
   - 置信度：XX%
   - 可证伪条件：如果[X]发生，说明护城河判断是错的

6. **输出**：护城河评估报告，存入 `10-研究报告输出/`
