# Moat Workflow — 护城河工作流

适用：`@护城河 [公司]`

---

## 步骤

1. **数据准备**：ROIC趋势（近5年）、毛利率趋势、市占率变化、定价能力证据

2. **巴菲特护城河9问**
   - 是否拥有无形资产（品牌/专利/许可）？持续性如何？
   - 是否有转换成本（客户粘性）？量化流失率
   - 是否具备网络效应？用户增长→价值增长的飞轮
   - 是否有成本优势（规模/流程/地理）？
   - 是否有高效规模（有限市场的自然垄断）？
   - 护城河是在拓宽还是收窄？（关键趋势问题）
   - 管理层是否在合理分配资本？
   - 主要竞争威胁是谁？威胁程度？
   - 10年后这家公司的护城河还在吗？

3. **波特五力验证**
   - 新进入者威胁 / 替代品威胁 / 买方议价力 / 供方议价力 / 行业竞争强度

4. **ROIC分析**
   - ROIC > WACC = 创造价值
   - MROIC（边际ROIC）趋势：新增投资回报率是否上升？

5. **护城河评级**
   - 宽护城河 / 窄护城河 / 无护城河
   - 置信度：XX%
   - 可证伪条件：如果[X]发生，说明护城河判断是错的

6. **输出**：护城河评估报告，存入 `10-研究报告输出/`
