# IndustryAnalysis Workflow — 行业格局与产业链分析 v1.0

适用：`@行业 [行业名]`、`@产业链 [行业名]`、`@格局 [行业名]`

⚠️ **定位说明**：本技能提供**行业级坐标**。个股层面的竞争优势用 `@护城河`，宏观层面用 `@宏观`。本技能可在 DeepAnalysis 之前执行，为个股分析提供行业背景。

## When to Use

Use when the user requests:
- "分析XX行业" / "@行业 XX" / "@产业链"
- "这个行业值得投吗" / "行业格局怎么样"
- "上下游分析" / "竞争格局"

**Do NOT use if:**
- 用户问某家公司的护城河 → 使用 `@护城河`
- 用户问宏观经济周期 → 使用 `@宏观`
- 用户需要个股深度分析 → 使用 `@分析`

---

## ⚠️ CRITICAL CONSTRAINTS

❌ **NEVER** 在未确认行业边界的情况下开始分析（必须先明确细分行业定义，避免过宽/过窄）  
❌ **NEVER** 跳过波特五力直接给出竞争格局结论（五力必须逐项打分）  
❌ **NEVER** 用"竞争激烈"等模糊描述代替 CR3/CR5/HHI 等集中度数据  
❌ **NEVER** 忽略产业链上下游议价能力分析（上游卡脖子/下游分散是致命风险信号）  
❌ **NEVER** 凭记忆引用市场规模和增速（必须标注数据来源和时间）

**最低交付标准：**
- ✅ 行业画像（TAM + 增速 + 生命周期 + 政策态度）
- ✅ 波特五力量化打分（每力 1-5 分 + 依据）
- ✅ 竞争集中度（CR3/CR5/HHI + 趋势）
- ✅ 行业 ROIC 分布与差异度
- ✅ 产业链图谱（上中下游 + 议价能力 + 微笑曲线定位）
- ✅ 行业评分卡（综合评级）

## 阶段一：行业画像

**search_web 行业新闻扫描（硬性前置步骤）：**
- `search_web("[行业名] industry trends market size [year]")` — 市场规模和增速最新数据
- `search_web("[行业名] 政策 监管 最新 [year]")` — 产业政策动向
- `search_web("[行业名] 竞争格局 并购 [year]")` — 竞争对手动态、新进入者/退出者
- → 提取结果用于下方各维度分析的最新素材

### 1.1 市场规模与增速

| 指标 | 数据要求 |
|---|---|
| **TAM（Total Addressable Market）** | 全球/中国市场规模，标注来源和年份 |
| **CAGR（历史）** | 过去 5 年复合增速 |
| **CAGR（预测）** | 未来 3-5 年预测增速 + 来源 |
| **渗透率** | 当前渗透率 vs 成熟市场标杆 |

### 1.2 行业生命周期定位

| 阶段 | 特征 | 投资含义 |
|---|---|---|
| **导入期** | 渗透率 <10%，增速 >30%，玩家多但格局未定 | 高弹性高风险，难辨赢家 |
| **成长期** | 渗透率 10-40%，增速 15-30%，龙头初现 | 最佳投资窗口，跟踪份额变化 |
| **成熟期** | 渗透率 >40%，增速 <10%，格局稳定 | 关注护城河和分红，价值投资区 |
| **衰退期** | 增速 <0%，替代品威胁明确 | 回避或做空，除非有转型催化 |

### 1.3 政策态度扫描

| 政策维度 | 评估内容 | 信号 |
|---|---|---|
| 产业政策 | 是否列入国家战略/重点扶持？ | 顺风 🟢 / 中性 🟡 / 逆风 🔴 |
| 反垄断/监管 | 是否面临反垄断审查或行业整顿？ | 🟢 / 🟡 / 🔴 |
| 税收/补贴 | 税惠是否持续？补贴有无退坡？ | 🟢 / 🟡 / 🔴 |
| ESG/碳达峰 | 碳排放约束对行业的影响方向 | 🟢 / 🟡 / 🔴 |

## 阶段二：竞争格局分析

### 2.1 波特五力量化版

每一力评分 1-5 分（1=威胁极低，5=威胁极高）。

| 力量 | 评分 | 量化指标参考 | 当前评估 |
|---|---|---|---|
| **现有竞争者竞争强度** | /5 | CR3/CR5、价格战频率、行业ROIC离散度 | [依据] |
| **潜在进入者威胁** | /5 | 资本壁垒（最低投资额）、牌照/许可证要求、规模经济拐点 | [依据] |
| **替代品威胁** | /5 | 替代技术成熟度、替代品价格比、用户转换意愿 | [依据] |
| **供应商议价能力** | /5 | 关键原材料集中度、是否存在卡脖子环节 | [依据] |
| **买方议价能力** | /5 | 客户集中度、品牌溢价能力、转换成本 | [依据] |
| **总分** | **/25** | 越低越好：<10=优质赛道，10-15=中等，>15=困难行业 | |

### 2.2 竞争集中度

| 指标 | 计算方式 | 判定标准 |
|---|---|---|
| **CR3** | 前3名市场份额之和 | >60%=高集中（有利龙头）, <30%=碎片化 |
| **CR5** | 前5名市场份额之和 | 对照行业 |
| **HHI 指数** | Σ(各企业份额²) | >2500=高垄断, 1500-2500=中等, <1500=分散 |
| **集中度趋势** | 近5年 CR3 变动方向 | 上升=强者恒强, 下降=格局松动 |

### 2.3 行业 ROIC 分布

- 拉取行业前 5-10 家企业的近 5 年 ROIC
- 计算行业 ROIC 中位数、均值和标准差
- **判定标准**：
  - ROIC 中位数 > WACC → 行业整体有经济利润
  - ROIC 标准差大 → 竞争分化明显，龙头优势可持续
  - ROIC 标准差小且均低 → 全行业赚不到钱，回避

## 阶段三：产业链图谱

### 3.1 上中下游拆解

| 位置 | 分析维度 |
|---|---|
| **上游** | 关键原材料/零部件供应商集中度、成本占比、是否存在卡脖子 |
| **中游** | 制造/组装环节的规模效应、技术壁垒、产能利用率 |
| **下游** | 终端客户类型（B2B/B2C）、渠道结构、品牌向下游传导能力 |

### 3.2 议价能力矩阵

| 环节 | 议价能力 | 依据 | 风险/机会 |
|---|---|---|---|
| 上游→中游 | 强/中/弱 | [供应商集中度、替代性] | [如：原材料涨价挤压中游毛利] |
| 中游→下游 | 强/中/弱 | [品牌力、产品差异化] | [如：渠道库存积压风险] |
| 下游→终端 | 强/中/弱 | [客户粘性、转换成本] | [如：大客户流失风险] |

### 3.3 微笑曲线定位

确定目标公司/行业在"微笑曲线"上的位置：

```
利润率
  高 │  研发/品牌        服务/渠道
     │    \               /
     │      \           /
  低 │        \       /
     │          \   /
     │           V
     │      制造/组装
     └───────────────────────→ 价值链
```

- **曲线两端（高利润）**：拥有知识产权/品牌的上游 + 拥有渠道/服务的下游
- **曲线底部（低利润）**：标准化制造/组装环节
- **判定**：公司位于曲线哪个位置？有无向两端攀升的趋势？

## 行业评分卡

| 维度 | 权重 | 评分 | 评级标准 |
|---|---|---|---|
| 市场空间与增速 | 25% | /25 | TAM大+高增速=满分 |
| 竞争格局友好度 | 25% | /25 | 波特五力总分<10=满分 |
| 产业链议价能力 | 20% | /20 | 所在环节议价强+无卡脖子风险=满分 |
| 政策与监管 | 15% | /15 | 纯顺风=满分 |
| ROIC 质量 | 15% | /15 | 行业ROIC中位数>15%且分化小=满分 |
| **总分** | 100% | **/100** | |

**总分判定**：
- 80+：⭐ 顶级赛道，值得深度覆盖
- 60-79：✅ 良好赛道，选择性覆盖龙头
- 40-59：⚠️ 一般赛道，需看个股Alpha
- <40：❌ 困难赛道，回避或仅做空
