# 踩烂策略（Stomp the Rotten）— 做空框架详解

## 概述

**踩烂策略**是风和基金针对做空策略提炼的独特方法论。核心思想：有些已经变烂的企业，还会变得更烂。

**胡猛形象比喻**: "有些已经变烂的企业，还会变得更烂，我们要做的就是踩上去，一直踩到它烂透为止。"

---

## 踩烂策略核心逻辑

### 为什么踩烂策略有效

**做空数学**:
```
股票从 100 元跌到 50 元再做空到 10 元：
- 空头收益：(50-10)/50 = 80%
- 确定性：高（右侧交易）

试图在 100 元猜顶做空到 10 元：
- 理论收益：(100-10)/100 = 90%
- 确定性：低（左侧交易，泡沫可能持续吹大）

结论：多等 50% 下跌，损失 10% 收益，换取高确定性
```

**投资者行为**:
- 第一波下跌（估值压缩）后，很多投资者会抄底
- 原有空头也会回补盈利了结
- 造成股价短暂反弹
- 但基本面继续恶化，业绩验证后股价二次下跌

**踩烂时机**: 在基本面恶化被逐季验证的过程中，坚持做空并"踩烂"到底，不见好转不平仓。

---

## 踩烂 vs 宴席理论

胡猛将做空分为两大类型：

| 维度 | 踩烂策略 | 宴席理论 |
|---|---|---|
| **类型** | 基本面恶化右侧确认 | 泡沫破灭左侧布局 |
| **时机** | 趋势已现，顺势做空 | 狂热顶峰，逆向做空 |
| **依据** | 盈利下滑数据验证 | 估值/情绪极端 |
| **持有** | 踩到底，不见好转不平仓 | 宴席散场即平仓 |
| **风险** | 管理层反扑、被收购 | 泡沫持续吹大、逼空 |
| **成功率** | 高（右侧） | 低（左侧） |
| **使用频率** | 高（主要策略） | 低（机会稀少） |

**胡猛总结**: "结构性做空机会本就不多，通常要等待 T3 级别的长期逻辑变化。"

## 踩烂策略适用条件

### 7 项检查清单

```
踩烂策略 Checklist
================================================================================

✅ [ ] 基本面恶化趋势明确（至少 2 个季度验证）
   - 营收连续下滑
   - 利润率连续下滑
   - 业绩低于预期

✅ [ ] 股价已跌一波但估值压缩非盈利下滑
   - 第一波下跌主要是估值压缩（PE 下降）
   - 盈利尚未大幅下滑（或刚开始）
   - 静态 PE 看似便宜

✅ [ ] 未来 2-4 季度盈利将继续下滑（有依据）
   - 行业供需恶化
   - 成本上升无法转嫁
   - 竞争加剧价格战
   - 一次性损失将持续

✅ [ ] 静态 PE 看似便宜但动态 PE 将升高
   - 当前 PE（基于过去 12 个月）看似低
   - 但未来盈利下滑，动态 PE 将升高
   - "估值陷阱"

✅ [ ] 管理层无明显反扑能力/计划
   - 管理层过往记录一般
   - 无明显扭亏计划
   - 现金流紧张无力回购

✅ [ ] 标的无被收购价值/可能
   - 资产质量一般
   - 无明显协同效应
   - 行业整体不景气

✅ [ ] 做空时机在右侧（趋势已现）
   - 不猜顶
   - 等第一波下跌后
   - 等基本面验证恶化

通过项：[X/7]
→ 7/7: 强烈做空
→ 5-6/7: 标准做空
→ 3-4/7: 谨慎做空
→ ＜3/7: 不做空
================================================================================
```

## 踩烂策略执行流程

### Step 1：识别潜在目标

**来源**:
- 财报业绩低于预期
- 管理层指引下调
- 行业供需恶化
- 竞争对手暴雷（传染效应）
- 负面新闻报道

**筛选标准**:
```
初步筛选:
✅ [ ] 营收连续 2 季度下滑
✅ [ ] 利润率连续 2 季度下滑
✅ [ ] 股价从高点下跌＞30%
✅ [ ] 静态 PE 看似便宜（＜15x）
```

### Step 2：基本面恶化分析

**分析维度**:

1. **营收恶化**
   - 下滑幅度
   - 下滑原因（需求/竞争/价格）
   - 是否可逆

2. **利润率恶化**
   - 毛利率变化
   - 营业利润率变化
   - 净利率变化
   - 恶化驱动因素

3. **现金流恶化**
   - 经营现金流
   - 自由现金流
   - 现金储备

4. **资产负债表恶化**
   - 负债率上升
   - 应收账款增加
   - 存货积压

### Step 3：估值分析

**关键洞见**: 踩烂策略的核心是识别"估值陷阱"

**估值陷阱特征**:
- 静态 PE 看似便宜（基于过去 12 个月盈利）
- 但未来盈利将大幅下滑
- 动态 PE（基于未来盈利）将大幅升高
- 市场尚未充分反应盈利下滑

**分析模板**:

### Step 4：风险评估

**做空特有风险**:

| 风险类型 | 描述 | 评估方法 |
|---|---|---|
| **管理层反扑** | 优秀管理层可能力挽狂澜 | 评估管理层过往记录 |
| **被收购** | 低估公司可能被收购托底 | 评估资产价值/协同效应 |
| **逼空** | 空头拥挤可能引发逼空 | 检查空头头寸比例 |
| **时间成本** | 做空时间未必是朋友 | 评估恶化速度 |
| **政策干预** | 政府可能救助 | 评估系统重要性 |

**风险评估模板**:

### Step 5：目标价与止损

**目标价确定**:

方法 1：**估值法**
```
目标价 = 正常化 EPS × 合理 PE

其中:
- 正常化 EPS: 行业低谷期的盈利水平
- 合理 PE: 行业低谷期的平均估值
```

方法 2：**技术面法**
```
目标价 = 历史支撑位

寻找:
- 上一轮周期低点
- 重要均线支撑
- 成交量密集区
```

**止损确定**:
```
止损价 = 入场价 × 1.20（+20%）

止损触发:
- 价格触及止损价
- 基本面好转信号（至少 2 个季度验证）
- 被收购公告
```

### Step 6：仓位管理

**做空仓位建议**:

| 确定性等级 | 通过检查项 | 仓位上限 |
|---|---|---|
| 高确定性 | 7/7 | 6%-8% |
| 中确定性 | 5-6/7 | 4%-6% |
| 低确定性 | 3-4/7 | 2%-4% |

**加仓策略**:
- 初始仓位：2%
- 验证恶化后：4% → 6%
- 最高不超过 8%

## 踩烂策略案例

### 案例：潍柴动力（2015 年踩烂做空）

**背景**: 2015 年 4 月以来股价已跌 50%，PE 跌到个位数

**踩烂分析**:

1. **基本面恶化**:
   - 工程机械行业需求持续下滑
   - 公司营收连续下滑
   - 利润率从峰值回落

2. **估值陷阱**:
   - 静态 PE 个位数，看似便宜
   - 但盈利将继续下滑
   - 动态 PE 将升高

3. **未来预测**:
   - 行业景气仍在恶化
   - 未来几个季度盈利将持续下滑
   - 股价还将下跌

4. **风险评估**:
   - 管理层优秀但无力回天（行业整体下行）
   - 无被收购可能（体量大）
   - 逼空风险低

5. **执行**:
   - 坚持空单不放
   - 不因静态 PE 便宜平仓
   - 目标价基于行业低谷估值

**结果**: 随后几个季度盈利持续下滑，股价又一波大跌，实现更大做空收益

## 踩烂策略常见误区

### 误区 1：把周期底部当踩烂

**区别**:
- **踩烂**: 结构性恶化，长期趋势向下
- **周期底部**: 周期性波动，未来将反转

**识别方法**:
- 行业是结构性衰退还是周期波动？
- 竞争格局是否永久恶化？
- 需求是暂时下滑还是永久消失？

### 误区 2：过早入场猜顶

**错误做法**:
- 在股价还在上涨时做空
- 试图抓住最高点
- 与趋势为敌

**正确做法**:
- 等第一波下跌后
- 等基本面验证恶化
- 右侧交易

### 误区 3：因估值便宜平仓

**错误做法**:
- 看到静态 PE 便宜就平仓
- 被"估值陷阱"迷惑
- 过早结束空头头寸

**正确做法**:
- 看动态 PE 而非静态 PE
- 等基本面真正好转
- "除非基本面好转，否则绝不因估值便宜而平仓"

### 误区 4：忽视管理层反扑

**错误做法**:
- 低估企业家的主观能动性
- 忽视管理层可能的扭亏措施
- 被突然的利好消息打爆

**正确做法**:
- 评估管理层过往记录
- 关注现金流状况（是否有能力反扑）
- 设置止损，防范意外

## 注意事项

1. **做空需要耐心**: "做空更需要耐心等待进入时机"
2. **时间未必是朋友**: 与做多不同，做空有时间成本
3. **严格止损**: 触及止损线必须执行，不论基本面观点
4. **不做左侧**: 不预测顶部，等右侧信号
5. **分散做空**: 不要集中做空单一标的
6. **关注催化剂**: 做空需要催化剂（财报/事件）
