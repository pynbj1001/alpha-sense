# 5M 分析框架详解

## 概述

5M 框架是胡猛利润率投资体系的核心分析工具，围绕 ROE（净资产收益率）这一核心，将公司基本面分析分解为五个维度：

1. **M1: Market Size（目标市场）**
2. **M2: Market Share（市场份额）**
3. **M3: Margin（利润率）**
4. **M4: Model（商业模式）**
5. **M5: Management（管理团队）**

---

## M1: Market Size（目标市场）

### 评估内容

- **TAM（总可服务市场）规模**: 公司所面对的市场容量
- **市场增速**: 行业增长率（CAGR）
- **周期位置**: 行业处于生命周期的哪个阶段

### 数据来源

- 行业研究报告（Gartner、IDC、Frost & Sullivan 等）
- 公司财报中的市场描述
- 行业协会数据
- 政府统计数据

### 评估标准

| 等级 | TAM 规模 | 行业增速 | 评估 |
|---|---|---|---|
| 优秀 | ＞$100B | ＞15% | 大市场 + 高增长 |
| 良好 | $50B-$100B | 10%-15% | 中等市场 + 中速增长 |
| 一般 | $10B-$50B | 5%-10% | 小市场 + 低速增长 |
| 较差 | ＜$10B | ＜5% | 小市场 + 停滞 |

### 分析要点

1. **市场边界定义**: 明确公司实际服务的市场范围
2. **细分市场结构**: 识别高增长细分 vs 衰退细分
3. **市场驱动因素**: 什么在推动市场增长/萎缩
4. **周期性特征**: 市场是周期性还是结构性变化

---

## M2: Market Share（市场份额）

### 评估内容

- **当前市场份额**: 公司在目标市场中的占有率
- **份额变化趋势**: 提升/稳定/下降
- **竞争格局**: 市场集中度、主要竞争对手

### 计算方法

```
市场份额 = 公司营收 / 目标市场规模
```

### 数据来源

- 公司财报（营收数据）
- 行业报告（市场规模数据）
- 竞争对手财报（用于对比）
- 第三方调研（渠道份额数据）

### 评估标准

| 等级 | 市场份额 | 趋势 | 评估 |
|---|---|---|---|
| 优秀 | ＞30% | 持续提升 | 市场领导者 + 扩张 |
| 良好 | 15%-30% | 稳定或提升 | 有力竞争者 |
| 一般 | 5%-15% | 稳定 | 跟随者 |
| 较差 | ＜5% | 下降 | 边缘化 |

### 分析要点

1. **份额提升的驱动因素**: 
   - 产品竞争力
   - 渠道扩张
   - 并购整合
   - 竞争对手退出

2. **份额提升的可持续性**:
   - 是否依赖补贴/价格战
   - 是否存在天花板
   - 竞争对手反击能力

3. **市场份额与利润率的权衡**:
   - 有时份额提升以牺牲利润率为代价
   - 需要评估长期战略合理性

---

## M3: Margin（利润率）— 核心要素

### 评估内容

- **毛利率**: (营收 - 营业成本) / 营收
- **营业利润率**: 营业利润 / 营收
- **净利率**: 净利润 / 营收
- **利润率趋势**: 历史变化与未来预测

### 胡猛核心洞见

**利润率变化是最富弹性的变化来源**:

```
差公司弹性：利润率从 1% → 10% = 利润增长 10 倍
好公司弹性：利润率从 10% → 20% = 利润增长 2 倍
```

因此，**利润率投机**是赚取"Delta"的核心策略。

### 历史区间分析

**步骤**:

1. 收集至少 5 年（最好 10 年）的季度/年度利润率数据
2. 计算历史最大值、最小值、中位数
3. 确定当前利润率在历史区间的位置（百分位）
4. 判断当前是低估还是高估

**评估标准**:

| 历史百分位 | 情绪判断 | 策略含义 |
|---|---|---|
| ＜20% | 极度悲观 | 潜在买点（做多） |
| 20%-40% | 悲观 | 谨慎乐观 |
| 40%-60% | 中性 | 观望 |
| 60%-80% | 乐观 | 谨慎悲观 |
| ＞80% | 极度乐观 | 潜在做空点 |

### 利润率驱动因素

**内部因素**:
- 产品结构升级（高毛利产品占比提升）
- 规模效应（固定成本摊薄）
- 效率提升（运营杠杆）
- 成本管控

**外部因素**:
- 行业供需关系
- 原材料价格波动
- 竞争格局变化
- 定价能力变化

### 利润率预测框架

```
未来利润率 = 当前利润率 + Δ(内部因素) + Δ(外部因素)

关键问题:
1. 行业供需未来 12 个月如何变化？
2. 公司成本结构有何变化？
3. 产品组合是否优化？
4. 竞争格局是否改善？
```

## M4: Model（商业模式）

### 评估内容

- **赛道属性**: 成长/周期/防御
- **风口位置**: 强风/微风/无风/逆风
- **收入结构**: 各业务线占比与增速
- **盈利模式**: 如何赚钱（一次性/经常性）

### 赛道分类

| 类型 | 特征 | 估值特点 | 代表行业 |
|---|---|---|---|
| **成长型** | 高增速、高投入 | 高 PE、高波动 | 科技、生物医药 |
| **周期型** | 随经济周期波动 | PE 波动大 | 航空、大宗商品、半导体 |
| **防御型** | 稳定增速、抗周期 | 低 PE、低波动 | 公用事业、必选消费 |

### 风口判断

**胡猛洞见**: "站在风口上，猪都会飞"

| 风口位置 | 特征 | 策略 |
|---|---|---|
| **强风** | 行业高增长、资本涌入、估值高企 | 警惕泡沫，准备做空 |
| **微风** | 行业稳健增长、理性投资 | 做多优质公司 |
| **无风** | 行业停滞、资本撤离 | 寻找结构性机会 |
| **逆风** | 行业衰退、普遍亏损 | 做空/寻找反转机会 |

### 收入结构分析

**要点**:
1. 各业务线营收占比
2. 各业务线增速对比
3. 各业务线利润率对比
4. 收入结构变化趋势

**警示信号**:
- 核心业务增速放缓
- 低毛利业务占比提升
- 一次性收入占比过高
- 客户集中度过高

## M5: Management（管理团队）

### 评估内容

- **CEO/CFO 背景**: 教育、工作经历、过往业绩
- **危机处理记录**: 过往应对危机的表现
- **股东导向**: 增持/减持、分红/回购政策
- **资本配置能力**: 投资、并购、研发投入决策

### 评估维度

| 维度 | 优秀标准 | 警示信号 |
|---|---|---|
| **能力** | 行业资深、成功记录 | 缺乏相关行业经验 |
| **诚信** | 信息披露透明、无违规 | 财务造假、违规记录 |
| **股东导向** | 大股东增持、高分红回购 | 频繁减持、低分红 |
| **稳定性** | 核心团队稳定 | 高管频繁离职 |

### 资本配置评估

**优秀配置**:
- 高 ROIC 的再投资
- 估值低时回购股票
- 战略性并购（协同效应明确）
- 适度分红

**糟糕配置**:
- 低 ROIC 的盲目扩张
- 估值高时增发股票
- 破坏价值的并购（商誉减值）
- 不分红也不回购（现金闲置）

### 管理层访谈要点

**关键问题**:
1. 对未来 12-24 个月行业趋势的判断
2. 公司核心竞争力是什么
3. 最大风险因素是什么
4. 资本配置优先级（研发/并购/回购/分红）

## 5M 综合评估

### 权重分配

胡猛经验：在风和基金前十年，**M1（市场）和 M4（风口）**是主要收益来源；未来环境下，**M2（份额）、M3（利润率）、M5（团队）**更重要。

**当前推荐权重**:
- M1 目标市场：15%
- M2 市场份额：20%
- M3 利润率：30%（核心）
- M4 商业模式：15%
- M5 管理团队：20%

### 评分标准

| 总分 | 评级 | 策略 |
|---|---|---|
| 4.5-5.0 | 强烈推荐 | 重仓做多（6%-8%） |
| 3.5-4.4 | 推荐 | 标准做多（4%-6%） |
| 2.5-3.4 | 中性 | 轻仓或观望（0%-2%） |
| 1.5-2.4 | 谨慎 | 轻仓做空（2%-4%） |
| 1.0-1.4 | 强烈做空 | 标准做空（4%-6%） |

### 鹰猪分析

结合 5M 评分进行鹰猪分类：

| 类型 | 5M 特征 | 策略 |
|---|---|---|
| **鹰** | M2↑M3↑M5 强 | 做多核心标的 |
| **猪** | M1 大 M4 热，但 M2/M3/M5 弱 | 做空/回避 |
| **风** | 纯靠行业 Beta，无 Alpha | 做空行业/ETF |

## 案例研究

### 案例 1：东方航空（周期反转做多）

**背景**: 2015 年航空业低谷，全行业亏损

**5M 分析**:
- M1: 航空市场需求稳定，长期增长 ✓
- M2: 市场份额稳定，行业集中度高 ✓
- M3: 利润率处历史底部，全行业亏损 → 拐点将至 ✓
- M4: 强周期行业，处于低谷 → 上升期 ✓
- M5: 管理层一般，但周期向上时都能盈利 ✓

**选择逻辑**: 财务差的公司（东航）弹性大，利润率从微利→正常水平，增幅数倍

**结果**: 行业周期向上，东航盈利改善幅度远超优质公司，股价大幅上涨

### 案例 2：潍柴动力（踩烂做空）

**背景**: 2015 年股价已跌 50%，PE 个位数看似便宜

**5M 分析**:
- M1: 工程机械市场需求下滑 ✗
- M2: 份额稳定但市场萎缩 ✗
- M3: 利润率从峰值下滑，还将继续下滑 ✗
- M4: 周期见顶向下 ✗
- M5: 管理层优秀但无力回天 ✗

**踩烂判断**: 行业景气仍在恶化，盈利将继续下滑，静态 PE 看似便宜但动态 PE 将升高

**结果**: 随后几个季度盈利持续下滑，股价又一波大跌

## 注意事项

1. **5M 不是机械打分**: 需要深入理解每个 M 背后的驱动因素
2. **动态调整**: 5M 评分应随时间和新信息更新
3. **重点在 M3**: 利润率变化是最富弹性的，应投入最多研究精力
4. **结合情绪面**: 5M 是基本面，需结合情绪面判断时机
5. **能力圈原则**: 只做自己理解的 5M 分析，不碰看不懂的
