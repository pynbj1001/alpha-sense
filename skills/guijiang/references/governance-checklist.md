## 二、"明股实债"深度排查（归江特别警示）

> 归江明确指出："一些公司通过成立子公司，向内部人或关联方发行带有高额固定利息的优先股。这种结构在财报上看似股权投资，实质却是高息债务。"

### 检查步骤

**Step 1：子公司与参股公司排查**
- 查阅合并报表附注中所有受限资产说明
- 识别持有优先股股东的性质（是否为内部人/关联方）
- 查找"固定回报""优先分配""托管协议"等关键词

**Step 2：高息安排识别**
- 股东协议中是否存在固定回报条款（如"保证年化8%回报"）
- 是否存在回购承诺或资产抵押安排

**Step 3：破产清算优先级**
- 若公司清算，这些"股东"是否享有优先偿付权（实际上是"债权人"）
- 普通公众股东在清算顺序中是否处于最末位

**识别关键词**（年报/公告中出现即需深查）：
```
优先股 / 优先分配 / 固定回报 / 保底收益 / 托管协议 /
名股实债 / 股权回购 / 估值调整机制（VAM）
```

| 风险等级 | 触发条件 |
|---|---|
| 🔴 一票否决 | 确认存在明股实债安排，普通股东利益受损 |
| 🟠 需解释 | 存在上述关键词但未确认，需管理层解释 |

## 三、关联交易审查

### 检查维度

| # | 检查项 | 正常范围 | 风险信号 |
|---|---|---|---|
| 3.1 | 关联方销售占总收入比例 | <10% | >30%，且无商业合理解释 |
| 3.2 | 关联方采购占总成本比例 | <15% | >30%，定价不公正 |
| 3.3 | 向关联方提供贷款/担保 | 轻微或无 | 大额且无信用支撑 |
| 3.4 | 关联方占款（回款异常慢） | 账期正常 | 应收账款中关联方长期欠款 |
| 3.5 | 关联交易定价独立性 | 第三方独立审计确认 | 关联方审计师自行确认 |

### "隧道行为"识别模型

以下组合出现时，高度警惕大股东通过关联交易侵占上市公司利益：

```
大股东持股>60%
+ 大量关联采购（价格高于市价）
+ 大量关联销售（价格低于市价）
+ 公司利润下行/现金流紧张
= 隧道行为高度疑似
```

## 五、财务质量排查

### 信息披露透明度评分

| 评分项 | 满分 | 评分标准 |
|---|---|---|
| 分部信息详细程度 | 20分 | 主要业务线的收入/利润分拆是否清晰 |
| 重要会计政策说明 | 20分 | 收入确认/资产减值等政策是否透明 |
| 风险因素披露质量 | 20分 | 是否有实质性风险描述而非套话 |
| 投资者问题回应质量 | 20分 | 互动平台/投资者交流会回复是否有实质内容 |
| 历史预测准确度 | 20分 | 过去3年管理层指引与实际业绩的偏差程度 |

**总分解读**：
- 80-100分：信披质量优秀
- 60-79分：可接受，部分领域需主动核实
- 40-59分：信披质量差，需多方交叉验证
- <40分：严重不透明，慎重

### 复杂成本核算预警

归江特别提示：以下行业成本核算透明度较低，需更深的草根调研：

| 行业 | 风险点 |
|---|---|
| 中药 | 原材料（虫草/人参/藏红花等）成本难核实，价格波动大 |
| 白酒（部分） | 虽原料相对简单，但年份酒定价可操纵 |
| 工程/建筑 | 分包利润难跟踪，关联方工程易出现利益输送 |
| 军工 | 成本单一客户，定价不透明 |
| 文化/娱乐 | 内容制作成本易虚增 |

## 历史典型案例（A股治理风险案例库）

> 本节为学习参考，不针对具体公司作投资建议。

### 案例类型一：明股实债导致普通股东巨损

常见结构：上市公司设立子公司→子公司向控股股东关联实体发行优先股→优先股约定固定回报8-12%→子公司出现问题→普通股东在清算中排最后位

**识别教训**：必须逐一审查合并报表附注中的子公司股权结构。

### 案例类型二：中药原材料成本操控

常见结构：虚增原材料采购价格→成本虚增→利润调节空间大→财报失真

**识别教训**：对比同行业同类产品的原材料单价；实地草根调研供应商。

### 案例类型三："隧道行为"典型路径

常见结构：大股东资金紧张→指令上市公司向关联方采购（高于市价）→关联方资金流回大股东→上市公司利润下降→小股东受损

**识别教训**：追踪关联方采购价格合理性，与第三方价格比较。
