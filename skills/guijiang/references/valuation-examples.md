# 三重保守估值计算示例与模板

> 归江："估值不是为了追求一个精确的数字，而是为了确立一个保守的价值区间，并在此基础上要求足够大的安全边际。"

**核心原则**：三种方法均指向低估，才具备安全边际。仅有一种方法支撑的"便宜"是危险的。

---

## 方法一：股息贴现模型（DDM）

### 适用场景

✅ 最适用于：
- 现金牛类成熟企业（FCF稳定，股息政策清晰）
- 过去3-5年连续分红、派息比率稳定的公司
- 与归江"现金回报新基石"主题契合的标的

⚠️ 不适用于：
- 成长期企业（股息低或不分红）
- 周期性企业（盈利波动导致股息预测困难）
- 亏损企业

### 公式

$$V = \frac{D_1}{k - g}$$

其中：
- $D_1$ = 预期下一年度每股股息
- $k$ = 要求回报率（归江保守取：10年国债收益率 + 风险溢价）
- $g$ = 股息长期可持续增长率

### 归江保守化调整原则

| 参数 | 常规取值 | 归江保守调整 |
|---|---|---|
| $k$（要求回报率） | 10年国债 + 3% | 10年国债 + **4-5%**（多留余量） |
| $g$（长期增长率） | 历史增速 | 历史增速 × **0.6-0.7**（打七折） |
| $D_1$（预期股息） | 分析师一致预期 | 取预期的**保守下限** |

### 三情景计算框架

```
悲观情景（权重 40%）：
  k = 10年国债 + 5%, g = 历史g × 0.5
  V_悲观 = D₁ / (k_高 - g_低)

中性情景（权重 40%）：
  k = 10年国债 + 4%, g = 历史g × 0.65
  V_中性 = D₁ / (k_中 - g_中)

乐观情景（权重 20%）：
  k = 10年国债 + 3%, g = 历史g × 0.8
  V_乐观 = D₁ / (k_低 - g_高)

综合保守DDM估值 = V_悲观 × 40% + V_中性 × 40% + V_乐观 × 20%
```

### 计算示例（假设数据）

**假设条件**：
- 公司过去3年平均每股股息：0.80元
- 股息增长历史：年均8%
- 当前10年国债收益率：2.5%

| 情景 | k | g | D₁ | 内在价值 |
|---|---|---|---|---|
| 悲观 | 7.5% | 4.0% | 0.83 | 23.7元 |
| 中性 | 6.5% | 5.2% | 0.83 | 62.3元 |
| 乐观 | 5.5% | 6.4% | 0.83 | 即分母趋0，需谨慎 |

**注意**：极端乐观情景下分母趋近于零，DDM失效，此时应主要依靠另外两种方法。

**综合DDM估值**：保守取悲观+中性加权均值，作为参考区间下限。

## 方法二：重置成本法（Replacement Cost）

### 适用场景

✅ 最适用于：
- 重资产传统行业（工厂/设备/基础设施）
- 需要评估资产底线价值（防止"价值陷阱"另一面的高估）
- 归江将其视为"极度保守的估值锚"

⚠️ 不适用于：
- 轻资产/知识密集型企业（品牌/软件/平台）
- 资产已严重过时或折旧的行业

### 公式

$$V_{重置} = 有形资产重建成本 + 无形资产折算价值$$

### 计算步骤

**Step 1：有形资产重建成本**
```
固定资产重建成本 = 净资产账面值 × 重建成本系数
（重建成本系数：考虑通胀调整，一般取 1.1-1.3的当前市价比）
存货现值 = 存货账面值（通常可直接用）
客户关系重建成本 = 估算重新获取同等客户基础的销售费用
```

**Step 2：无形资产折算**
```
品牌价值估算：
  - 方法A：品牌溢价定价法（如能以高于仿制品X%定价，则品牌价值 = X% × 年销售额 × 品牌寿命年数 / 折现率）
  - 方法B：简化法：取过去5年累积广告/品牌建设费用的残值估算

专利/技术价值：
  - 核心专利数量 × 行业平均单项专利综合价值（可参考同行业并购交易的技术溢价倍数）

许可证/准入资质：
  - 行业新进入壁垒高度 × 替代成本（申请周期 + 资金投入）
```

**Step 3：安全边际标准**
```
归江安全边际要求：市值 < 重置成本 × 0.8
（即相对于重建成本具有20%以上折价）
```

### 计算示例（假设数据）

**假设：某重型装备制造企业**

| 资产类别 | 账面值（亿元） | 重建成本系数 | 重建成本（亿元） |
|---|---|---|---|
| 厂房/设备 | 50 | 1.2 | 60 |
| 存货 | 10 | 1.0 | 10 |
| 客户关系 | — | — | 8（估算重建销售费用） |
| 品牌价值 | — | — | 5（累积广告费残值） |
| 技术/专利 | — | — | 3 |
| **重置成本合计** | | | **86** |

归江安全边际下，买入价格上限：86 × 0.8 = **69亿元**

若当前市值为55亿，安全边际 = (69-55)/69 = **20%**，达到归江最低要求。

## 方法三：盈利能力价值法（EPV）

### 适用场景

✅ 最适用于：
- 盈利稳定但增长温和的成熟企业
- 归江关注"可持续正常盈利"而非峰值盈利
- 用于检验DDM估值的一致性

⚠️ 注意：
- 必须正态化处理（剔除异常项）
- 必须验证FCF与净利润的一致性（防止利润虚高）

### 公式

$$EPV = \frac{EBIT_{正态化} \times (1-t)}{WACC}$$

其中：
- $EBIT_{正态化}$ = 近3-5年正态化税前息前利润（剔除一次性项）
- $t$ = 归江保守取有效税率（而非名义税率）
- $WACC$ = 加权平均资本成本（归江偏取偏高值）

### 正态化 EBIT 步骤

```
Step 1：获取过去5年EBIT

Step 2：识别并剔除异常项（每年）：
  - 一次性资产处置收益/损失
  - 商誉减值
  - 政府补贴（非经常性）
  - 诉讼赔偿/赔偿收入
  - 灾害损失

Step 3：加权平均
  - 建议权重：近年 > 远年（如：近2年各30%，第3年30%，第4-5年各5%）

Step 4：对正态化EBIT进行保守下调（10-15%估算误差留量）
```

### WACC 的保守取值

```
WACC = Kd × (1-t) × D/V + Ke × E/V

归江保守化规则：
1. 无风险利率：取当前10年国债 + 0.5%（周期性上行保护）
2. 股权风险溢价（ERP）：取 5.5-6%（A股历史上限）
3. Beta：若不确定，取 1.0-1.1（保守偏高）
4. 负债成本：取实际融资成本 + 0.5%（未来利率风险）
```

### 计算示例（假设数据）

**假设：某消费品企业**

| 年份 | 报告EBIT（亿元） | 剔除项 | 正态化EBIT |
|---|---|---|---|
| T-4 | 8.2 | -资产出售收益1.5 | 6.7 |
| T-3 | 9.1 | 无 | 9.1 |
| T-2 | 10.3 | -政府补贴0.8 | 9.5 |
| T-1 | 11.5 | -商誉减值1.0 | 10.5 |
| T | 12.0 | 无 | 12.0 |

加权平均（近3年：70%；T-4/T-3：各15%）：
```
正态化EBIT = 12.0×0.30 + 10.5×0.25 + 9.5×0.25 + 9.1×0.10 + 6.7×0.10
           = 3.6 + 2.625 + 2.375 + 0.91 + 0.67 = 10.18
保守下调10%：10.18 × 0.9 = 9.16亿元
```

假设 t=25%，WACC=8%:

$$EPV = \frac{9.16 \times (1 - 0.25)}{0.08} = \frac{6.87}{0.08} = 85.9亿元$$

## 常见估值误区警示

| 误区 | 正确做法 |
|---|---|
| 使用单一估值方法下结论 | 三种方法均验证 |
| 使用峰值盈利计算EPV | 必须正态化处理 |
| 使用乐观增长假设计算DDM | 必须使用保守下限 |
| 市值填充（用目标市值推反算增长率） | 先独立估值，后对比市价 |
| 不做压力测试 | 必须计算悲观情景下安全边际 |
| 依靠记忆数据 | 所有数字实时获取，标注来源 |
