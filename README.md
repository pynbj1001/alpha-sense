## 🚀 快速开始（3步）

### Step 1：打开文件夹
用 VS Code 打开本文件夹 → AI 自动加载 `.github/copilot-instructions.md` 系统指令

### Step 2：开始对话
按 `Ctrl+L`（或点击 Copilot Chat）打开 AI 聊天窗口

### Step 3：输入指令
```
@分析 美团        ← 生成个股深度研究报告
@估值 腾讯        ← 三重估值 + 市赚率分析
@行业 AI芯片      ← 行业研究报告
@宏观             ← 宏观环境 + 资产配置
@芒格 AI泡沫？    ← 以芒格视角回答
```

报告自动输出到 `10-研究报告输出/` 目录。

## 🧠 核心原则

1. **安全边际优先** — 价格远低于价值时买入
2. **能力圈自律** — 不懂的不碰
3. **概率化表达** — 所有结论用概率区间，拒绝"一定"
4. **逆向检查** — 每个结论都做 Pre-mortem
5. **多框架验证** — 至少 3 个框架交叉确认
6. **简洁有力** — 一句话说不清就不该买

## 🔧 高级用法

### 自然语言提问
不记快捷指令也没关系，直接用自然语言提问：
```
"帮我用格林沃尔德三重估值分析贵州茅台"
"当前处于康波什么阶段？对资产配置有什么建议？"
"用10X Alpha框架看看哪些A股有十倍潜力"
```

### 指定框架组合
```
"用巴菲特护城河 + 市赚率 + 贝叶斯拐点三个框架分析比亚迪"
```

### 更新知识库
将新文件放入对应模块目录，AI 自动识别。建议同时导出 .txt 版本以提高读取效率。
