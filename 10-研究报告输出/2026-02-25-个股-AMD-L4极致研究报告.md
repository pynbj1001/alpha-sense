## 执行摘要（3句话版本）

**结论：** AMD的FY2025财务数据展现出真实的基本面拐点——营收同比增34%至$34.6B、FCF爆增180%至$6.7B、数据中心GPU从0到快速规模化——这是真正的业务转型而非报表粉饰。

**估值：** 前向PE 19.7x对于一家EPS同比增长217%的公司并不贵，但DCF基准情景显示内在价值约$130-220美元，当前价格已充分定价基础情景，过度依赖MI400 AI GPU的持续超预期交付。

**行动：** 对长期投资者（3年+），当前价格对应约10-15%年化回报的可接受风险/回报比；短期（1年内）若遇回调至$160-180区间（约Forward PE 15-17x），将出现显著更优的买入机会。

## 四、十大框架全流派分析

### 框架1：格林沃尔德三重估值（资产→EPV→成长价值）

**A. 资产价值（Asset Value）**

| 资产类别 | 账面价值 | 调整后值 | 说明 |
|---------|---------|---------|------|
| 净有形资产 | ~$18B | ~$18B | 所有者权益$63B - 商誉$24B - 无形资产~$21B |
| 产品IP/研发资本化 | — | +$15B | 累计研发投入的现值（内部资产） |
| 净现金 | +$6.6B | +$6.6B | |
| **调整后资产价值** | | **~$40B** | |
| **每股资产价值** | | **~$24.5** | 股价$213 >> 资产价值 |

> **结论：** 资产法不适用于AMD这类轻资产高增长公司（商业模式价值远超账面资产）。股价是资产价值的8.7倍，正常。

**B. 盈利能力价值（EPV）**

```
EPV = 调整后正常化收益 / 资本成本

正常化EBIT：FY2025 EBIT $36.9亿（含大量Xilinx摊销约$30亿/年，需加回）
  + 无形资产摊销（非现金）  ~$28亿
  ≈ 调整后EBIT ~$65亿
税率 ~15%（AMD有效税率较低，享有税收抵免）
  → 调整后NOPAT = $65 × (1-15%) ≈ $55亿

资本成本 WACC = 9.5%（无风险 4.3% + 市场溢价6% × Beta 0.85→调整Beta）

EPV = $55亿 / 9.5% = $579亿 = $579B
每股EPV = $579B / 16.3亿股 ≈ $355/股
```

> **EPV结论：** 当前股价$213 < EPV $355，意味着市场对AMD的**正常化盈利能力定价偏低**，存在约40%的EPV折价。这被框架1.C的成长价值部分解释——市场实际上在为成长潜力支付价格，而大量摊销拉低了报告盈利。

**C. 成长价值（Growth Value）**

```
成长价值 = EPV × 成长溢价倍数

可持续ROE（有形）= ~24%
WACC = 9.5%
成长溢价倍数 = ROE/WACC = 24%/9.5% ≈ 2.5x

AMD具备护城河（AMD会在下方护城河框架中分析），给予成长价值溢价
→ 内在价值估算 = EPV × 1.5-2.0x =  $530 - $710亿... → ~$130-175/股
```

但这是保守估算，未纳入数据中心AI GPU的非线性增长。纳入增长期权后：
- **格林沃尔德内在价值区间：$150 - $260/股**
- **当前$213 = 中间情景**，基本公平

**置信度：65%｜核心假设：FY2026E有形NOPAT达$70亿+，AMD维持护城河**

## 八、TSR分解与盈亏平衡分析

### 8.1 TSR（总股东回报）分解

**TSR = 盈利增速g + 股息率d + 估值变动Δ(PE)**

| 时段 | 盈利增速g预期 | 股息 | PE变动预期 | **预期TSR** |
|------|-------------|------|----------|------------|
| 1年（FY2026） | +45-55% EPS增长 | 0% | -10至0%（PE收缩） | **+35-55%** |
| 3年（FY2028） | 年化~30% | 0% | -20%（PE from 20→15x） | **年化15-20%** |
| 5年（FY2030） | 年化~20% | 开始分红? | ±0% | **年化15-20%** |

### 8.2 盈亏平衡增长率（g*）

```
当前PE = 19.7x（前向）
股息率 = 0%
g* = 1/PE - 股息率 = 1/19.7 = 5.1%

这意味着：市场(AMD前向PE 19.7)实际上只要求AMD盈利增长5.1%就能获得零超额收益
→ AMD目前盈利增速217%（追踪）/ 预期FY2026E ~50%
→ 远大于g*=5.1%
→ 这意味着AMD估值对于增长实现的"要求门槛极低"!
```

> **关键洞察：** 前向PE仅19.7x的"要求门槛"极低（g*=5.1%），而实际EPS增速可能达50%+，这提供了巨大的盈利超预期空间。这一框架强烈支持AMD的多头论点。

## 十一、二阶效应与反身性分析

### 11.1 正向反身性链条

```
AMD AI GPU规模扩大

→ [正反馈闭环，类似CUDA历史]
```

### 11.2 负向反身性链条（最需警惕）

```
AI CapEx泡沫破裂（云厂商ROI不如预期）
→ Microsoft/Meta削减MI300订单
→ AMD季报miss → 股价大跌（-30-40%）
→ AMD融资成本上升 → 研发投入受限
→ MI400竞争力减弱 → 市场份额进一步流失
→ [负螺旋]
```

> **反身性最大风险：** AI投资回报（ROI）的验证周期约18-24个月。如果到2026年底企业AI项目没有展示出清晰的ROI（成本降低/收入增加），云厂商可能大规模削减AI GPU订单。这将同时打击NVDA和AMD，但AMD由于生态护城河更薄弱，受冲击将更严重。预估此风险概率：25%。

### 11.3 非线性拐点识别

**"如果ROCm生态突破，AMD将出现非线性加速"**

当ROCm支持的模型/框架数量达到CUDA的70%，开发者的迁移成本将从"难以接受"变为"可以接受"，届时AMD的AI GPU市场份额获取速度可能出现**非线性加速**（从15%加速到25-30%，而非线性外推的20%）。这个拐点的时间节点：估计2027-2028年，概率40%。
