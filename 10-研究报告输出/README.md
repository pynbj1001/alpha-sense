# 10-研究报告输出

> **本目录是所有 AI 研究报告的输出位置。**

## 命名规范

```
[日期]-[类型]-[标的]-[框架].md
```

### 类型代码
| 类型 | 代码 | 说明 |
|------|------|------|
| 个股深度 | `个股` | 单一公司全面分析 |
| 行业研究 | `行业` | 行业格局与趋势 |
| 宏观策略 | `宏观` | 宏观环境与资产配置 |
| 决策备忘录 | `决策` | 买入/卖出/调仓决策 |
| 对比分析 | `对比` | 两家以上公司对比 |
| 快速笔记 | `笔记` | 简短灵感与观察 |

### 示例
```
2026-02-08-个股-美团-多框架综合.md
2026-02-08-行业-AI芯片-技术革命框架.md
2026-02-08-宏观-康波定位-周金涛框架.md
2026-02-08-决策-买入腾讯-备忘录.md
```

## 可用模板

| 模板 | 适用场景 |
|------|----------|
| `模板-个股深度研究报告.md` | 个股全面分析 |
| `模板-行业研究报告.md` | 行业研究 |
| `模板-宏观策略报告.md` | 宏观与配置 |
| `模板-投资决策备忘录.md` | 买卖决策记录 |

## 使用方式

在 VS Code 聊天中输入快捷指令（如 `@分析 美团`），AI 将自动使用对应模板生成报告到本目录。
