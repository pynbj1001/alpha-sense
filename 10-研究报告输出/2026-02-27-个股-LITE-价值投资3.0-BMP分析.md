## PHASE 3: 财务报表调整

### 3.1 原始财务数据 (FY2025) — **已修正**

| 研发费用 | $303.9M | yfinance income_stmt |
| SG&A | $348.2M | yfinance income_stmt |
| 毛利率 | 28.0% | (459.9/1645) |
| 稀释股数 | ~69.8M | yfinance balance_sheet |

**关键发现**: LITE FY2025实际**营业利润为负**，公司处于亏损状态。这与info中显示的"10.7%营业利润率"存在巨大差异，可能是数据年份不同（info可能使用不同财年或TTM数据）。

### 3.2 调整逻辑

**问题**: 公司为何亏损？
- R&D + SG&A = $303.9M + $348.2M = $652.1M
- 营收 $1,645M → 毛利率28% → 毛利润 $460M
- 毛利润 $460M - 运营支出 $652M = **-$192M 营业亏损**

**解读**: 高强度的研发和营销投入是亏损主因，但这些投入是为了未来的增长。

### 3.3 稳态利润计算

**调整思路**: 假设公司进入成熟期，研发和营销降至行业正常水平

| 科目 | FY2025实际 | 调整假设 | 调整后 |
|-----|-----------|---------|--------|
| 营收 | $1,645M | 不变 | $1,645M |
| 毛利率 | 28% | 维持 | 28% |
| 毛利润 | $460M | - | $460M |
| R&D | $303.9M | 降至15%营收 | $247M |
| SG&A | $348.2M | 降至12%营收 | $197M |
| 调整后营业利润 | -$192M | - | **$16M** |

**保守稳态假设** (参考成熟光模块公司利润率):
- 稳态营业利润率: 15%
- 稳态利润: $1,645M × 15% = $247M
- 潜在净利润: $247M × (1-21%) = $195M

### 3.4 潜在EPS估算

```
稀释股数 = 69.8M

潜在EPS (保守) = $195M / 69.8M = $2.79
```

## PHASE 4: 估值分析 (P)

### 4.1 前瞻P/E计算 — **已修正**

**参数假设**:

| 参数 | 假设 | 依据 |
|-----|------|------|
| 收入CAGR | 30% (保守) | AI光模块需求旺盛 |
| 稳态利润率 | 15% | 参考成熟光模块公司 |
| 税率 | 21% | 当前税率 |

**3年后预测**:

```
当前营收: $1,645M

3年后营收 = $1,645M × (1.30)^3 = $3,637M

稳态利润 = $3,637M × 15% = $545M
潜在净利润 = $545M × (1-21%) = $431M

潜在EPS = $431M / 69.8M = $6.17
```

### 4.2 估值结果 — **已修正**

```
当前股价: $677
潜在EPS (3年后): $6.17

调整后前瞻P/E = $677 / $6.17 = 109.7x
```

**对比报表估值**:
- 报表P/E (196x): 基于$3.45 EPS (TTM)
- 实际FY2025 EPS: $25.9M / 69.8M = **$0.37**
- 报表P/E = $677 / $0.37 = **1,829x** (天价)

### 4.3 估值决策

| 估值指标 | 数值 | 评估 |
|---------|------|------|
| 报表P/E (基于TTM $3.45) | 196x | 极高 |
| 实际P/E (基于FY2025 $0.37) | 1,829x | 不具参考性 |
| 调整后前瞻P/E | **109.7x** | **极高** |
| 目标P/E阈值 | 20x | 未达标 |

**决策**: ❌ **REJECT (拒绝)**

**理由**:
- 调整后前瞻P/E高达110x，远超20x阈值
- 公司FY2025实际营业亏损$192M
- 当前股价完全建立在未来高增长预期上
- 任何预期落空都将导致股价大幅回调
