3.3 对投资论点的影响
正面验证：
- Gemini 3.0 Deep Think 在 IMO-ProofBench Advanced 达到 90% 正确率（2026-01）——说
明 Google 的 AI 研究确实处于前沿
- 从”chatbot”到”agentic AI”的范式转换意味着每用户的推理算力需求将数量级跃升——这为
大规模 CapEx 提供了需求侧的技术逻辑
- Deep Think 的 B2B 科研应用（药物发现、工程优化）直接对接 Google Cloud 的企业客户——
Cloud backlog $240B 的增长动力可追溯至此
风险因素：
- 技术领先不等于商业领先——OpenAI o1/o3、Anthropic Claude 也在快速追赶推理能力

- Agentic AI 的变现模型尚未成熟——用户是否愿意为 Agent 付费，定价几何？
- Deep Think 目前仅向 AI Ultra 订阅用户和 API 研究者开放——规模化变现时间表不确定
四、为什么市场在害怕？我来说说我的独立判断

（分析师共识 ~$115B）
2. FCF 将被毁灭性压缩 → 2026E FCF 从 $73.3B 骤降至 ~$13B → FCF yield 从 1.96% 降至

4. MROIC 边际衰减 → 投资翻倍后，新增 $90B 的边际回报率是否仍 >> WACC？历史上所有

我的独立判断

3. $240B backlog 是最硬的需求验证：4.1x 年收入覆盖 = 未来 4 年的收入可见性极高

五、多框架并行分析
框架 A：MROIC 边际资本回报率分析
问题：新增的 $60B CapEx（净 CapEx 从 $20B→$70B）能产生足够回报吗？
计算：
- 2022→2025 新增 OCF = $164.7B - $91.5B = $73.2B
- 2022→2025 累计净 CapEx = $18.0B + $20.4B + $37.2B + $70.3B = $145.9B
- 简化 MROIC = $73.2B / $145.9B = ~50%

即使用粗略计算，MROIC 远高于 WACC（~9-10%）。Google 的增量资本投资正在创造巨大价
值。
但要注意：
- 大部分 OCF 增长来自广告业务的自然增长 + 运营杠杆，不能全部归因于 CapEx
- AI/Cloud 相关 CapEx 的独立 MROIC 更难衡量
- 真正的检验在 2026-2028：Cloud 收入增速能否从 ~30% 加速？AI 变现能否规模化？
结论：MROIC 目前健康，但边际回报的可持续性需要密切观察。
置信度：70%
可证伪条件：如果 2026 年 Cloud 增速降到 20% 以下且 AI 收入占比不到 10%，说明 MROIC 在恶
化。

框架 C：技术革命指数引擎分析
AI 基础设施是否遵循莱特定律？
莱特定律预测：累计产量每翻一倍，成本下降固定百分比
Google TPU 已到第六代，每代性价比提升 2-4x
早期超额投资创造成本优势 → 一旦曲线走通，后来者难以追赶
类比：亚马逊 2010-2018 在 AWS 上的激进投资→建立了不可逆的规模优势

指数引擎检验：
- 搜索广告：线性增长（非指数）——但 AI 提升 CPC 效率
- Cloud：指数级增长潜力——但高竞争
- AI 模型/API：处于 S 曲线早期——爆发力最大但不确定性最高
- Waymo 自动驾驶：极长周期期权
结论：CapEx 中的一部分（Cloud + AI 模型）具有指数引擎特征，但搜索广告的 CapEx 回报是线
性的。
置信度：60%
二阶思考：”然后呢？”——如果 Google 在 AI 基础设施上建立类似 AWS 的规模优势，其他玩家
将被迫退出或并入生态，形成正反馈循环。但反面是：如果 AI 被证明是 commoditized 的，所有
玩家的 MROIC 都会下降。

十二、二阶效应备注
1. D&A 海啸效应：2025 CapEx $91.4B + 2026E $180B = 累计 $271B 的新增资产。假设 5 年直
线折旧，2027-2028 年 D&A 将从 $21B 飙升至 $60-75B/年。这会将 OPM 从 32% 压缩至
~22-25%，但 after-tax cash flow 反而因 $15-20B/年的额外税盾而增加。市场可能过度惩罚
利润率下降而忽略现金流改善。

2. GPU 贬值与自研芯片对冲：2026 CapEx 中 60%（~$108B）投向服务器/GPU——面临 2 年一
代的技术迭代贬值风险。但 Google 的 Ironwood TPU 自研芯片战略可部分对冲对 NVIDIA
GPU 的依赖和定价压力。
3. Agentic AI 的算力乘数效应：Gemini 3.0 从”对话式 AI”转向”Agent 式多步执行”——每
次 Agent 任务的算力消耗可能是简单对话的 10-100x。这意味着即使用户数不增长，算力需求
也可能指数级上升——为 $180B CapEx 提供了技术需求侧的非线性论据。
4. 监管反身性升级：$175-185B 的 CapEx 规模本身就是一种”护城河宣示”——这个数字会让监
管机构更加警觉。投资越成功 → AI 市场地位越强 → 反垄断压力越大 → 可能被迫开放 API/数据
或分拆。
5. 回购机会成本：2025 回购 $45.7B → 2026 在 FCF ~$13B 的约束下，回购可能降至 $15-25B。
以当前 PE 28.6x 计算，$20B 回购 vs $20B CapEx 的 ROI 对比：回购的回报率 = 1/PE = 3.5%
（确定性），CapEx 的 MROIC = 待验证。每多投 $1B 在 CapEx 而非回购上，就是一个隐性
的”MROIC > 3.5%”赌注。
6. Gemini 3.0 Deep Think 的非线性价值：如果 Deep Think 的科研能力真的能加速药物发现/材
料科学——这不是线性价值而是平台性价值。类似 AWS 之于云计算，Google 可能成为全球科
研算力的基础设施供应商——TAM 远超当前 Cloud 收入。但这是 5-10 年的期权，不是 2026
的收入。
