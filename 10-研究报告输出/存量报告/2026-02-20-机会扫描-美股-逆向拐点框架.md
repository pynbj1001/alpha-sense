### 🥇 #1 UBER — 优先级：🔥高

**一句话论点**：市场因Waymo/特斯拉Robotaxi叙事而惩罚UBER，但UBER本身就是自动驾驶时代最大受益者——它不需要造车，只需要继续做全球最大出行平台。

| 预期PE | 17.1x |

| 毛利率 | 38.5% |
| 营收增速 | 20.1% |
| FCF收益率 | **4.19%** ← 极为罕见（高FCF + 低估） |
| 市赚率PR | **0.39** ← 低于0.5，极度低估 |
| 市值 | $1,516亿 |

**框架分析**

- **卡拉曼安全边际**：FCF yield 4.19% + PE15倍，对应隐含的盈利增速预期约0%。但UBER实际营收增速20%，利润改善持续——**安全边际大于30%（高置信度80%）**
- **贝叶斯拐点（逆向压力∝机会大小）**：市场共识认为Waymo会吃掉UBER → 历史上每次"平台颠覆恐惧"估值压缩，UBER反而加速部署自动驾驶合作（Waymo已入驻UBER平台）→ 市场定价的颠覆风险被高估，概率更新：**UBER作为AV时代平台的胜率：65%-75%**
- **护城河**：网络效应（司机-乘客双边飞轮）+ 地理多元化 + 品牌即品类，护城河评分：**4/5**

**可证伪条件（失效信号）**：
- 失效信号1：若Waymo独立APP月活超过UBER月活的20%且UBER市占率下滑>5pp — 则论点失效
- 失效信号2：若营收增速连续2个季度低于12% — 则重估

**二阶思考**：UBER一旦成为AV调度平台（类似安卓+手机的关系），其网络效应反而比现在更强、边际成本更低 → 盈利改善将超预期 → 戴维斯双击可能性（利润+估值同时修复）较高
