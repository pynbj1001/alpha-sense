一、公司画像：Cloudflare 是什么？

1.1 一句话定义

Cloudflare 是一家边缘云基础设施公司，通过遍布全球 330+ 城市的数据中心网络，为企业和开发者提供安全、性能和网络即服务。本质上是"互联网的中间层"——在用户与源服务器之间，拦截攻击、加速内容、执行计算逻辑。

二、商业模式深度解析

2.1 产品矩阵（四大支柱）

支柱一：应用服务（Application Services）

  • CDN/缓存：全球分发静态与动态内容
  • DNS：全球最快的 1.1.1.1 解析器 + 托管 DNS
  • 负载均衡 & 智能路由：Argo Smart Routing
  • Web 优化：自动压缩、图片优化、等待室

支柱二：安全（Security）

  • DDoS 防护：自动化、无限容量 DDoS 缓解
  • WAF （Web 应用防火墙）：机器学习驱动
  • Bot 管理：识别并拦截恶意机器人
  • API 安全：API Gateway + Schema Validation
  • SSL/TLS：全站加密
  • Email 安全：收购 Area 1 后整合

支柱三：Zero Trust / SASE（网络即服务）

  • Cloudflare One：整合 ZTNA + SWG + CASB + DLP
  • Cloudflare Access：Zero Trust 访问控制
  • Cloudflare Gateway：安全 Web 网关
  • Magic WAN/Transit：广域网即服务，替代传统 MPLS
  • 远程浏览器隔离（RBI）

支柱四：开发者平台（Developer Platform）

  • Workers：边缘 Serverless 计算
  • R2：S3 兼容对象存储（零出站费用）
  • D1：边缘 SQLite 数据库
  • Workers AI：边缘 AI 推理（GPU 加持）
  • AI Gateway：AI 应用的缓存/限流/可观测性
  • Pages / Stream / Images：前端托管与媒体服务

3.2 Cloudflare 在价值链中的位置——"互联网的中间层"

关键洞察：Cloudflare 占据了所有互联网流量必经的"交通枢纽"位置。

根据 W3Techs 数据，Cloudflare 代理了互联网上约 20% 的网站流量，是全球最大的反向代理网络。这意味着：

  • 所有经过 Cloudflare 的流量都可以被观察、分析、优化、保护
  • 网络效应：更多流量 → 更好的威胁情报 → 更好的安全 → 吸引更多客户
  • 数据飞轮：每秒处理 5700万+ HTTP请求，形成了全球最大的互联网威胁数据集

3.4 产业拼图关键判断

"微笑曲线"上的位置

Cloudflare 处于价值链的中游偏上游——不是最终应用（SaaS），而是让所有应用都更快、更安全、更便宜运行的基础设施层。这类似于互联网的"水电煤"。

关键卡位优势

  1. 流量必经：反向代理模式意味着所有 HTTP(S) 请求都经过 Cloudflare 的网络，自然延伸到安全→计算→存储
  2. 通用性极强：不绑定任何特定云厂商或行业，为所有下游场景服务
  3. 产品粘性递增：客户从 CDN 开始，叠加 WAF → Zero Trust → Workers → R2，每增加一个产品，迁移成本指数上升

利润迁移方向

当前利润主要在 安全 和 CDN 层（成熟产品，高毛利）。未来利润增量将来自： - Zero Trust / SASE：TAM $30B+，Cloudflare 仍在渗透初期 - 开发者平台：Workers + R2 + D1 构成的边缘计算生态 - AI 推理：Workers AI 将 GPU 推理部署到边缘，TAM 潜力巨大但极早期

五、护城河分析（巴菲特框架）

5.1 护城河类型评估

护城河类型 强度 证据
网络效应 ⭐⭐⭐⭐ 代理 20% 互联网流量 → 最大威胁情报数据集 → 更好安全 → 更多客户。典型的数据飞轮。
转换成本 ⭐⭐⭐⭐ 客户叠加 CDN+WAF+Zero Trust+Workers 后，迁移成本极高。DNS 切换影响全站可用性。
规模效应 ⭐⭐⭐⭐ 330+ 城市数据中心，固定网络成本摊薄，边际流量几乎零成本。新进入者无法复制。
品牌/信任 ⭐⭐⭐ 企业安全信任建立需时，Cloudflare 在 DDoS 防护领域已是第一选择。
成本优势 ⭐⭐⭐ 自建网络 + 免出站费（R2）颠覆 AWS S3 定价，但主业并非成本竞争。

5.2 护城河宽度判断

估计新进入者要复制 Cloudflare 当前网络需要投入 $15-20B+ 资本，且需 5+ 年建设期。即便有资本，缺乏流量规模的威胁情报数据集是不可复制的。

EPV/资产价值比（定性推断）：Cloudflare 的 EPV（基于正常化后的未来盈利能力估算）将显著高于其有形资产重置价值（$810M PP&E + 网络），说明存在明确的特许经营权价值。

护城河存在概率：70-75%

5.3 护城河风险（Pre-mortem）

如果护城河判断错误，最可能的原因是： 1. AWS/Azure/GCP 捆绑打击：三大云厂商将安全+CDN+边缘计算免费/低价捆绑 2. Zero Trust 竞争加剧：Zscaler、Palo Alto 等原生安全厂商在 SASE 领域更强 3. 技术替代：如果 WebAssembly 边缘计算模式被其他范式替代 4. 中国/地缘风险：Cloudflare 需处理内容审核争议，可能被某些市场封禁

7.2 盈利能力价值（EPV）— 零增长假设

标准化盈利推算（假设零增长，当前收入可持续）： - 当前收入：$2,013M - 标准化毛利率：76%（$1,530M） - 标准化费用（零增长下削减增长相关投入）： - R&D 加回 30%（增长相关）：~$490M × 70% = $343M - S&M 加回 25%（增长相关）：~$700M × 75% = $525M
- G&A 保持：~$100M - 标准化营业利润：$1,530M - $343M - $525M - $100M = $562M - 税后（假设 21%）：$444M - 扣除维持性 CAPEX（$150M）：$294M - 折现率 R = 10% - EPV（企业价值）= $294M / 10% = $2,940M → 每股 ~$8.4

EPV >> 有形资产价值（$3.85），说明明确存在特许经营权/护城河。

EPV - 资产价值 = $4.55/股 → 护城河价值

但 EPV $8.4/股 远低于当前股价 $180（更不用说盘后 $208），说明市场对成长价值给予了极高溢价——当前股价中约 95% 来自成长期望。

7.3 成长价值

如果 Cloudflare 能实现以下假设： - 未来 5 年收入 CAGR 25%（从 $2B → $6.1B） - 终态营业利润率 20%（参考成熟 SaaS 公司） - 终态净利率 15%（$915M） - 5 年后给予 30x PE（高增长 SaaS），得到终态市值 ~$27.5B - 折现回当前（10% 折现率）：~$17B → 每股 ~$49

加上 EPV $2.9B → 保守内在价值 = $17B + $2.9B ≈ $20B → ~$57/股

如果假设更乐观（收入 CAGR 28%，终态利润率 25%，终态 PE 40x）： - 终态市值 ~$62B - 折现 → ~$38.5B → 每股 ~$110

估值区间：$57 - $110/股（中枢 ~$80/股）

当前收盘价 $180 → 溢价 64-216%，安全边际严重不足。

⚠️ 重要说明：盘后 $208 的价格可能反映了最新财报超预期（FY2025 Q4），但即便如此，当前估值已将未来 5-8 年的增长完全定价。
