## 执行摘要

基于格林沃尔德三重估值框架，Uber的内在价值区间呈现**显著分层特征**：

- **资产价值(AV)**: $2.15–$5.20/股（平台轻资产特性，参考价值有限）

**核心结论**: EPV/AV = **9.3x**，远超2.0x强护城河阈值；当前股价$70.72已反映部分成长预期，但基准情景下仍存在**20-30%上行空间**。

## 一、资产价值（Asset Value）分析

### 1.1 调整逻辑

Uber作为典型的**轻资产双边平台**，其会计账面价值严重低估真实经济价值。格林沃尔德资产重置法从"竞争进入视角"评估：新进入者需要多少资本才能复制Uber的当前规模？

**关键调整科目**:

| 科目 | 账面值 | 调整方法 | 重置价值 | 调整说明 |
|------|--------|---------|---------|---------|
| 货币资金 | $7.63B | 保留收入1.5% | $0.78B | 超额现金$6.85B单独处理 |
| 应收账款 | $3.89B | 扣坏账准备5% | $3.70B | 平台收款周期短，风险低 |
| 固定资产 | $1.80B | 重置新价 | $2.50B | 服务器/办公设备重置成本 |
| 无形资产 | $2.85B | 重建成本法 | $4.50B | 10年R&D累积+品牌重建 |
| 商誉 | $2.25B | 保留（<20%阈值） | $2.25B | 商誉/净资产=9.1%，无Kill-Switch触发 |

### 1.2 三情景AV计算

```
资产重置总值 = $11.62B (基准)
(-) 运营性负债 = $4.46B
(-) 有息债务   = $6.30B
= 企业价值口径AV = $0.86B
(+) 超额现金   = $6.85B
= 股权AV       = $7.71B
÷ 稀释股本(2.058B股)
= AV每股(基准) = $3.75
```

| 情景 | 计算逻辑 | AV每股 | 适用场景 |
|------|---------|--------|---------|
| **悲观** | 清算折扣：流动资产×80%，固定资产×50% | **$2.15** | 行业衰退/平台解体 |
| **基准** | 重置成本法（上表） | **$3.75** | 正常竞争分析 |
| **乐观** | 含数据资产/品牌全额重建 | **$5.20** | 品牌价值显著场景 |

### 1.3 AV参考价值评估

**结论**: AV对Uber的定价参考价值**极低**。

原因：
1. 平台价值主要来自网络效应（用户+司机双边密度），非有形资产
2. $3.75/股的重置成本 vs $70.72市场价格，差距达19倍
3. 新进入者无法仅用$7.7B复制Uber的1.5亿年活跃用户和全球调度算法

**格林沃尔德视角**: AV在此仅作为"极端 downside"的底线参考，实际估值需依赖EPV和GV。

## 三、成长价值（Growth Value）分析

### 3.1 GV计算前提检查

| 前提条件 | 标准 | Uber状态 | 结果 |
|---------|------|---------|------|
| EPV/AV ≥ 1.3 | 有护城河迹象 | 9.3x ✅ | 满足 |
| Task 1护城河确认 | 定性支撑 | 双端网络效应 ✅ | 满足 |
| ROIC-WACC ≥ 5pp | 有效再投资 | 53.6pp ✅ | 满足 |

**结论**: 三项条件全部满足，**可以计算GV**。

### 3.2 自然增长参数

**行业自然增长率参考**:
- 全球共享出行市场CAGR: 8-12%
- Uber历史收入CAGR (FY21-25): 18%
- 保守假设自然增长 $g_1$ = 6% (GDP+2%)

**维持自然增长所需再投资**:
```
每元收入所需IC = 0.20x (平台轻资产特性)
年度再投资 = $52B收入 × 6% × 0.20 = $0.62B
```

### 3.3 价值创造因子（VCF）验算

```
近3年留存利润累计 = $9.76B FCF - $1.83B SBC + $6.52B回购调整 ≈ $14.5B
近3年NOPAT增量   = $6.70B - $2.76B = $3.94B
再投资回报率(ROIC) = $3.94B / $14.5B = 27.2%
VCF = 27.2% / 9% (WACC) = 3.0x
```

**判断**: VCF = 3.0x > 1.0x，**有效再投资**，留存利润创造价值。

### 3.4 GV三情景计算

成长价值公式: GV = EPV × (WACC/(WACC-g_eff) - 1)

## 五、关键假设清单（可证伪格式）

### 假设1: 正常化EBIT利润率
**取值**: 10.7% (FY2025) / 5年中位数3.0%  
**可证伪条件**: 若FY2026利润率低于5%且非一次性原因，则零增长假设失效，需下调EPV 15-20%。

### 假设2: 零增长调整加回比例
**取值**: SG&A 25%, R&D 25%  
**可证伪条件**: 若Uber宣布削减市场投入但单量增速低于5%，则加回比例高估，需下调EPV 5-10%。

### 假设3: WACC = 9%
**取值**: 无风险利率4.3% + 风险溢价5% × Beta 1.0  
**可证伪条件**: 若10年美债利率突破6%且维持1年以上，WACC需上调至10-11%，EPV降至$28-32。

### 假设4: 护城河期 = 7年（基准）
**取值**: 双端网络效应保护期  
**可证伪条件**: 若Waymo/Tesla在3年内于TOP10城市实现Robotaxi全覆盖且定价低于Uber 15%+，则护城河期缩短至3年，GV归零。

### 假设5: 有效增长率 $g_{eff}$ = 8%
**取值**: 行业增长6% + 运营杠杆2%  
**可证伪条件**: 若FY2026 GMV增速低于12%且利润率停止扩张，则增长率假设失效，含GV估值上限$60。

### 假设6: 价值创造因子 VCF = 3.0x
**取值**: 历史再投资ROIC 27% / WACC 9%  
**可证伪条件**: 若未来3年再投资ROIC降至WACC以下（即新投资项目无法覆盖资本成本），则GV归零，仅参考EPV。
