# Scoring Workflow

1. Pull latest quality, growth, price, and risk inputs with source/date.
2. Score Q-G-P-R on a 0-100 scale using the repository scoring rubric.
3. Adjust weights by company paradigm (mature, growth, cyclical, platform, turnaround).
4. Output total score, confidence interval, key drivers, and disconfirming signals.
5. Save to `10-研究报告输出/`.
