# TrapCheck Workflow

1. Pull latest financial and concentration data with source/date.
2. Run six negative-paradigm checks: multiple illusion, leverage-driven ROE, pseudo-growth, deep-value trap, consolidation-only growth, single dependency.
3. For each trap, assign risk level and evidence.
4. Apply kill-switch rules; if triggered, downgrade recommendation to Observe.
5. Save to `10-研究报告输出/`.
