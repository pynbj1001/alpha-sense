## 一、时代定位（每次执行前先确认）

执行本技能时，Agent **必须**先回答以下定位问题，并根据最新数据更新判断：

### 1.1 周金涛四周期仪表盘

| 周期 | 需判断的阶段 | 判断依据 |
|------|-------------|---------|
| **康波（50-60年）** | 萧条/回升/繁荣/衰退 | TFP增速、主导技术渗透率、全球利率趋势 |
| **库兹涅茨（20-25年）** | 底部/上升/顶部/下行 | 房地产价格、建设投资、人口结构 |
| **朱格拉（7-11年）** | 出清/复苏/扩张/过热 | 产能利用率、资本开支增速、信贷周期 |
| **基钦（3-4年）** | 主动去库/被动去库/主动补库/被动补库 | PMI库存分项、工业品价格、企业利润 |

### 1.2 佩雷丝技术周期定位

- AI革命处于「导入期/泡沫期/调整转折/部署期/成熟期」的哪个阶段？
- 当前金融资本与生产资本的关系如何？

### 1.3 达利欧长期债务周期

- 全球主要经济体处于「去杠杆/再杠杆」的哪个阶段？
- 是否出现"美丽的去杠杆化"条件？

## 二、六条投资铁律（任何时候不可违反）

> [!CAUTION]
> 以下铁律是**硬性约束**，任何投资建议必须全部通过才有效。

### 铁律1：生存第一
- 组合最大回撤必须控制在 **15-20%** 以内
- **禁止使用杠杆**（康波萧条/转折期的致命红线）
- 任何持仓必须能在 **1周内变现**（李蓓流动性规则）

### 铁律2：不怕踏空
- 风险收益比不佳时持有现金是**主动的正确决策**
- 看不到"四年三倍"空间的标的**不买**（王川标准）

### 铁律3：概率化表达
- 结论用概率区间，禁用"一定""肯定"
- **至少3个独立框架指向同一方向**才构成投资建议

### 铁律4：反向思考
- 别人贪婪时谨慎，别人恐慌时贪婪
- 最好的买点往往是最难受的时候（戴维斯双击时机）

### 铁律5：区分泡沫与价值
- 新技术的长期价值 ≠ 当前价格合理
- 泡沫破裂后的调整期恰恰是最佳买入窗口

### 铁律6：数据先行
- 禁止凭记忆引用财务数据，必须通过 yfinance/akshare/tushare 实时获取
- 标注数据来源与日期

## 三、标的筛选清单（7道关卡）

当用户要求评估某个投资标的时，**必须按顺序**通过以下7道关卡：

### 关卡1：周期适配检查 ⏱️
- [ ] 该标的所处行业在当前康波/朱格拉周期中是顺风还是逆风？
- [ ] 宏观流动性环境（央行政策方向）对该资产类别有利还是不利？
- **不通过条件**：逆周期 + 逆流动性 → 建议等待，不操作

### 关卡2：护城河验证 🏰
- [ ] 是否具有多维度垄断性？（规模/品牌/技术/生态/网络效应，至少3项）
- [ ] 竞争对手能否通过交叉补贴或快速跟进削弱其优势？
- [ ] 从用户/客户角度体验，是否具有不可替代性？
- **不通过条件**：护城河单一或存疑 → 降级为"观察"，不重仓

### 关卡3：临界点确认 📊
- [ ] 公司收入是否已超过 5-10亿美元？（王川临界点原则）
- [ ] 商业模式是否已跑通？（盈利或接近盈利）
- [ ] 是否已通过市场验证（而非仅靠概念/叙事）？
- **不通过条件**：未过临界点 → 仅允许试探性小仓位（<3%）

### 关卡4：估值合理性 💰
- [ ] 当前PE/PB相对于历史分位数处于什么位置？
- [ ] 是否存在"四年三倍"的空间？（年化≥30%）
- [ ] 做情景分析：乐观/中性/悲观三种情景下的预期回报
- [ ] 是否有戴维斯双击潜力（低估值 + 盈利即将超预期）？
- **不通过条件**：估值偏贵（历史>75分位）且无明确催化剂 → 等待回调

### 关卡5：地缘风险评估 🌐
- [ ] 该标的是否过度依赖单一国家/地区市场？
- [ ] 供应链是否受地缘冲突影响？
- [ ] 是否存在制裁/政策/战争等不可控风险？
- **不通过条件**：地缘风险高且无法对冲 → 降低配置权重

### 关卡6：仓位适配 ⚖️
- [ ] 加入该标的后，组合是否满足地域分散要求？（不超40%单一国家）
- [ ] 行业集中度是否可控？
- [ ] 整体现金/短债比例是否仍 ≥ 20%？
- **不通过条件**：导致组合过度集中 → 调整仓位至安全比例

### 关卡7：Kill-Switch 红线检查 🚨
以下任一项触发 → **立即降为"观察"，不投资**：
- [ ] 需要使用杠杆才能达到目标收益
- [ ] 公司存在财务造假嫌疑
- [ ] 管理层诚信存疑
- [ ] 单一客户依赖 > 30%
- [ ] 行业处于政策打压周期
- [ ] 所在国家发生战争/资本管制
- [ ] 自己无法真正理解该商业模式（能力圈外）

## 四、资产配置模板（乱世版）

### 4.1 战略配置框架

| 资产类别 | 目标权重 | 调整区间 | 核心逻辑 |
|---------|---------|---------|---------|
| **黄金/贵金属** | 15-25% | 10-30% | 货币信用对冲 + 地缘避险 + 康波萧条期优势资产 |
| **现金/短期国债** | 20-30% | 15-40% | 保持弹药 + 无风险收益 + 流动性储备 |
| **高股息防御** | 20-30% | 15-35% | 稳定现金流 + 低估值安全边际 |
| **AI核心资产** | 15-25% | 5-30% | 新康波主导产业 + 分层分批建仓 |
| **其他机动** | 5-10% | 0-15% | 周期性机会 + 新兴市场 |

### 4.2 AI资产分层配置

```
第一层：基础设施层（确定性最高，可较早配置）
├── 算力芯片龙头（通过护城河验证）
├── 云计算平台（已证明AI变现能力）
└── AI能源基础设施（数据中心电力需求）

第二层：平台层（中等确定性，等回调配置）
├── 有真实AI收入增长的平台巨头
└── AI工具链/中间件龙头

第三层：应用层（高不确定性，小仓位试探）
├── AI+垂直行业应用（医药/金融/制造）
└── 等待临界点——收入过5亿、商业模式跑通后再重仓
```

### 4.3 地域分散框架

| 地域 | 配置逻辑 | 权重范围 |
|------|---------|---------|
| **美国** | AI核心地，但注意高估值+政治风险 | 30-40% |
| **中国/港股** | 低估值+AI应用潜力，政策风险需对冲 | 15-25% |
| **其他/新兴市场** | 逆全球化"替代供应链"受益者 | 5-10% |
| **无国籍资产** | 黄金/大宗商品，跨越地缘风险 | 15-25% |

## 五、执行纪律

### 5.1 建仓规则
- **分批建仓**：任何标的至少分3批买入，间隔 ≥ 2周
- **越垄断越加仓**：随护城河确认加强而逐步增加仓位（王川原则）
- **回调加仓**：优先在AI板块回调20-30%时加仓，而非追高

### 5.2 卖出规则
- **让赢家奔跑**：护城河无恙 + 现金流稳定 → 持有
- **卖出触发条件**：①基本面恶化（护城河受损）②出现明显更好的机会
- **永远留一半**：减仓时至少保留50%核心仓位，避免完全踏空

### 5.3 定期审视（建议频率）
- **每周**：检查宏观流动性信号（央行声明、利率变化）
- **每月**：审视持仓护城河是否依旧成立
- **每季**：做一次全组合的情景分析和再平衡评估
- **每年**：更新四周期仪表盘定位（第一节）
