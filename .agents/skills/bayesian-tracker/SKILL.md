# 🎯 贝叶斯 Lollapalooza 追踪器技能 (Bayesian Tracker)

> **核心原理**：结合查理·芒格的 Lollapalooza 效应与贝叶斯定理。通过自动收集五大维度（价格、基本面、新闻、宏观、技术面）的证据信号，利用似然比（Likelihood Ratio）持续更新投资论点成立的后验概率。当概率突破临界点（如 ≥75% 买入，≤25% 卖出）时，系统会生成交易提醒。

---

## 🛠️ 工具使用指南（给 Agent 的指令）

当用户呼叫 `@贝叶斯 [公司代码]`，或要求你跟踪某个标的、更新概率时，你**必须**调用底层的 Python CLI 程序来实现追踪。所有状态修改都必须通过该 CLI 完成。

CLI 程序路径：`14.贝叶斯追踪器/tracker.py`

### 1. 常见场景及命令映射

| 场景 | 命令示例 | 适用情况 |
| --- | --- | --- |
| **添加新追踪** | `python 14.贝叶斯追踪器/tracker.py add --ticker NVDA --thesis "AI爆发" --target "翻倍" --prior 0.5` | 用户首次要求用贝叶斯跟踪某个标的时。你可以先让用户提供论点（thesis）和大目标（target）。 |
| **更新数据（核心操作）** | `python 14.贝叶斯追踪器/tracker.py run --ticker NVDA` | 当用户说“@贝叶斯 NVDA”或者要求更新/收集最新证据时。 |
| **查看追踪列表** | `python 14.贝叶斯追踪器/tracker.py list` | 用户想知道当前追踪了哪些标的，或者想全盘检视。 |
| **获取追踪历史** | `python 14.贝叶斯追踪器/tracker.py history --ticker NVDA` | 用户想查看某个标的的概率变化曲线或历史轨迹。 |
| **生成深度报告** | `python 14.贝叶斯追踪器/tracker.py report --ticker NVDA` | 用户要一份完整的追踪体检报告。会将报告输出并保存在 reports/ 下。 |
| **GUI 仪表盘** | `python 14.贝叶斯追踪器/tracker.py dashboard` | 弹出精美的 Tkinter 仪表盘（仅限有桌面环境时）。 |
| **全部更新** | `python 14.贝叶斯追踪器/tracker.py run-all` | 定期巡检所有活跃的追踪标的（可配合 Windows 定时任务）。 |
| **停用追踪** | `python 14.贝叶斯追踪器/tracker.py remove --ticker [标的]` | 用户说放弃跟踪或结束操作时。 |

### 2. 标准工作流 (Agent 执行流程)

如果用户输入 `@贝叶斯 AAPL`，请按以下流程执行：

1. **检查是否已在追踪列表中**：
   你可以先执行 `python 14.贝叶斯追踪器/tracker.py list`。如果列表中没有 AAPL，向用户询问：“请提供您跟踪 AAPL 的【投资论点】、【大目标】以及【初始发生概率 (默认50%)】。”获取后执行 `add` 命令。
   
2. **执行更新与信号收集**：
   如果列表中已有（或刚刚添加），直接执行 `python 14.贝叶斯追踪器/tracker.py run --ticker AAPL`。

3. **解读终端输出，组织回复内容**：
   阅读命令带来的终端输出，向用户作一份简明扼要的摘要：
   - 提取**当前概率**和**概率变化值**。
   - 提取**当前所处的阶段判断**（例如：🔥 扣动扳机 — 买入，或 📊 持续跟踪）。
   - 列举本次收集到的 **最有价值的重大信号**（特别是那些 Likelihood Ratio 偏离 1.0 较大的，即对改写概率起决定作用的信号）。
   - 如果发生突破临界点的情况（终端会高亮提醒），提醒用户考虑执行**交易动作**。

## 📊 五维度信号机制说明

1. **📈 价格动量 (Price)**：
   分析 52 周新高新低、MA200 位置、短期涨跌幅、成交量变动。
2. **📊 基本面 (Fundamental)**：
   分析 ROE 水平、营收增长率、毛利率、PE估值、流水比变动。
3. **📰 新闻情绪 (News)**：
   通过 ai-news-radar 和 stock_tracker 提取最新财报、新闻的情绪（Pos/Neg/Neutral）和关注度。
4. **🌐 宏观环境 (Macro)**：
   分析美国整体大环境如 VIX、美元指数、大宗商品（黄金原油）、流动性指标。
5. **📐 技术面 (Technical)**：
   利用 RSI(14) 超买超卖指标、MA20/MA50金叉死叉指标判断博弈点位。

所有的证据汇集后，通过 Lollapalooza 引擎的贝叶斯网络模型（对数似然比累积防浮点爆炸技术）完成新一轮“后验概率”的结算。

## 📉 阶段判断表

当概率落在不同区间时，程序会自动归位阶段，你也应当在语言中体现出此基调：

| 概率区间 | 阶段判断 | 你的话术建议 |
| :--- | :--- | :--- |
| **≥ 75%** | **🔥 扣动扳机 — 买入** | 强烈建议！Lollapalooza 共振已经形成，天时地利人和具备。 |
| **65% - 74%**| **📈 论点强化 — 加仓观察** | 继续向好！信号支持论点，可以寻找合适的技术位买入底仓。 |
| **45% - 64%**| **📊 持续跟踪 — 收集证据** | 局势尚不明朗。多空交织，需要耐心等待进一步实质性信号。 |
| **25% - 44%**| **⚠️ 论点动摇 — 警惕风险** | 势头不对！反面证据持续累积，需要审视原始论点是否已经破产。 |
| **< 25%** | **🔴 扣动扳机 — 卖出/回避**| 立刻防御！逻辑彻底被证伪，此时坚持即是偏执，建议换仓或止损。 |
