# Alpha Sense — 从"看对"到"大赚"

> 解决投资中最痛苦的问题：**看对了方向，但没赚到钱**

## 问题诊断

你的投研框架已是世界级的，但信号→信念→行动的管道出现了断裂：
- 产品直觉存在，但没有被系统捕获
- 信号捕获了，但没有持续追踪
- 追踪了，但信念不足以转化为仓位
- 仓位有了，但不够重

## 系统架构：6 阶段 Alpha 管道

```
📡 信号捕获 → 📊 贝叶斯追踪 → 🪜 信念升级 → 😤 遗憾复盘 → 🔭 AI猎手 → 📋 成果总览
```

每个阶段都有可视化面板、API 和数据持久化。

## 快速启动

```bash
cd "15.Alpha-Sense"
pip install -r requirements.txt
streamlit run app.py
```

浏览器打开 `http://localhost:8501` 即可使用。

## 6 个功能页面

| 页面 | 功能 |
|---|---|
| 📡 信号捕获 | 一句话输入 → 匹配标的 → 推入贝叶斯追踪器 |
| 📊 贝叶斯面板 | 全量概率更新、概率趋势图、维度拆解 |
| 🪜 信念阶梯 | 6级阶梯可视化、待升级提醒、升级确认 |
| 😤 遗憾复盘 | 录入遗憾、根因分析、防错规则生成 |
| 🔭 AI猎手 | 被忽略机会扫描、Watchlist覆盖率、周度复盘信 |
| 📋 成果总览 | 全部环节统计、数据导出 |

## 核心数据文件

- `data/signals.json` — 所有捕获的信号
- `data/regrets.json` — 遗憾案例
- `data/conviction_log.json` — 信念升级日志
- `data/weekly_reviews/` — 周度复盘信

## 与现有系统集成

- **贝叶斯追踪器** (`14.贝叶斯追踪器/`): 完全复用 TrackerStore/BayesianTracker
- **投资机会追踪** (`11.投资机会跟踪报告/`): 从 watchlist 扩展关键词匹配
- **教训档案** (`tasks/lessons.md`): 遗憾复盘自动追加
- **研究报告** (`10-研究报告输出/`): 成果总览统计

## 投研指令扩展

| 新指令 | 功能 |
|---|---|
| `@信号 [描述]` | 信号捕获 |
| `@遗憾 [标的]` | 遗憾复盘 |

## 信念升级阶梯

| 等级 | 概率区间 | 仓位 | 动作 |
|---|---|---|---|
| L0 信号捕获 | 0-55% | 0% | @信号 入库 |
| L1 快速扫描 | 55-60% | 0% | AI拉基本面快照 |
| L2 中度研究 | 60-65% | 1-3% | @估值 / @护城河 |
| L3 深度研报 | 65-70% | 5-8% | @分析 全套 |
| L5 买方决策 | 70-80% | 10-15% | @L5 CIO备忘录 |
| HEAVY 重仓 | 80%+ | 20-25% | @L6 沙盘 + @辩论 |
